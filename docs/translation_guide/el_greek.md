## 1. Τόνος και Επίπεδο

### Για το Synoptic Studio

- **Επίπεδο**: Επαγγελματικό αλλά προσιτό
- **Τόνος**: Σίγουρος, σαφής και βοηθητικός — ποτέ αλαζονικός
- **Φωνή**: Ενεργητική προτιμάται· παθητική μόνο για έμφαση στη δράση

### Βασικές αρχές

```
✅ «Αποθήκευσε το έργο σου» (άμεσο, ενεργητικό)
❌ «Το έργο σου θα αποθηκευτεί» (παθητικό)

✅ «Δημιούργησε όμορφα δίγλωσσα βιβλία»
```

---

## 2. Εσύ vs. Εσείς

### Κανόνας για το Synoptic

**Χρήση «εσύ»** — σύγχρονο πρότυπο για εφαρμογές τεχνολογίας.

| Πλαίσιο            | Μορφή     | Παράδειγμα                  |
| ------------------ | --------- | --------------------------- |
| Διεπαφή χρήστη     | **Εσύ**   | «Αποθήκευσε το έργο σου»    |
| Τεχνική τεκμηρίωση | **Εσύ**   | «Μπορείς να εξάγεις σε PDF» |
| Μηνύματα σφάλματος | **Εσύ**   | «Έλεγξε τη σύνδεσή σου»     |
| Νομικά κείμενα     | **Εσείς** | «Ο χρήστης αποδέχεται...»   |

### ⚠️ Σημείωση

- «Εσύ» είναι ανεπίσημο αλλά φιλικό
- «Εσείς» είναι πιο επίσημο
- Για τεχνολογία, το «εσύ» είναι η νόρμα

## 3. Ουδέτερη Γλώσσα ως προς το Φύλο

### Πρόκληση στα ελληνικά

Τα ελληνικά έχουν γραμματικό γένος (αρσενικό, θηλυκό, ουδέτερο).

### Συνιστώμενες στρατηγικές

| Μέθοδος                 | Παράδειγμα                  | Χρήση         |
| ----------------------- | --------------------------- | ------------- |
| **Πληθυντικός**         | «Οι χρήστες μπορούν»        | ✅ Φυσικό     |
| **Αναδιατύπωση**        | «Όποιος/Όποια χρησιμοποιεί» | ✅ Κομψό      |
| **Διπλός τύπος**        | «Χρήστες και χρήστριες»     | Επίσημο       |
| **Ουδέτερο ουσιαστικό** | «Άτομα που χρησιμοποιούν»   | ✅ Προτιμάται |

### Παραδείγματα

```
✅ «Καλώς ήρθες!» (ουδέτερο από το συμφραζόμενα)
✅ «Το έργο αποθηκεύτηκε» (παθητικό — ουδέτερο)

✅ «Για συγγραφείς και επιμελητές»
❌ «Καλώς ήρθες, αγαπητέ χρήστη»
```

## 4. Παγίδες Μετάφρασης

### Ψευδόφιλοι

| Αγγλικά     | ❌ Λάθος μετάφραση | ✅ Σωστή μετάφραση       |
| ----------- | ------------------ | ------------------------ |
| Actually    | Πραγματικά         | Στην πραγματικότητα      |
| Eventually  | Τελικά             | Τελικά ✓ / Κάποια στιγμή |
| Sympathetic | Συμπαθητικός       | Συμπονετικός             |
| Pathetic    | Παθητικός          | Αξιοθρήνητος             |

### Η παγίδα «Ultimate»

Στα αγγλικά, «ultimate» σημαίνει **«το καλύτερο, το απόλυτο»**.

```
✅ «Το απόλυτο δίγλωσσο στούντιο» — σωστή σημασία
✅ «Το καλύτερο δίγλωσσο στούντιο»
❌ «Το τελευταίο δίγλωσσο στούντιο» (υποδηλώνει τέλος)
```

## 5. Γραμματική και Σύνταξη

### Δομή πρότασης

Τα ελληνικά ακολουθούν γενικά **SVO** (Υποκείμενο-Ρήμα-Αντικείμενο):

```
Αγγλικά: I save the project (SVO)
Ελληνικά: Εγώ αποθηκεύω το έργο (SVO)
```

### Πτώσεις

Τα ελληνικά έχουν 4 πτώσεις:

| Πτώση      | Ερώτηση    | Παράδειγμα |
| ---------- | ---------- | ---------- |
| Ονομαστική | Ποιος; Τι; | το έργο    |
| Γενική     | Τίνος;     | του έργου  |
| Αιτιατική  | Ποιον; Τι; | το έργο    |
| Κλητική    | —          | έργο!      |

### Άρθρα

- Οριστικό: ο, η, το (οι, οι, τα)
- Αόριστο: ένας, μία, ένα

## 6. Τυπογραφία

### Τόνοι

Τα ελληνικά χρησιμοποιούν μόνο τον οξεία (´) στη σύγχρονη γραφή:

- **ά, έ, ή, ί, ό, ύ, ώ**

### ⚠️ Σημαντικό

Πάντα βάζε τόνους — αλλάζουν τη σημασία!

```
✅ «αποθήκευση»
❌ «αποθηκευση» (χωρίς τόνο)
```

### Εισαγωγικά

- Ελληνικά εισαγωγικά: **« »** (guillemets)
- Εναλλακτικά: **" "**

```
✅ «Το έργο σου "Το βιβλίο μου" αποθηκεύτηκε»
```

### Αριθμοί και μορφές

| Τύπος      | Μορφή      | Παράδειγμα |
| ---------- | ---------- | ---------- |
| Χιλιάδες   | Τελεία     | 1.000      |
| Δεκαδικά   | Κόμμα      | 3,14       |
| Νόμισμα    | X €        | 12,99 €    |
| Ποσοστό    | X% ή X %   | 50%        |
| Ημερομηνία | ΗΗ/ΜΜ/ΕΕΕΕ | 30/12/2025 |
| Ώρα        | ΩΩ:ΛΛ      | 14:30      |
