# ไทย (th) — คู่มือการแปล

## ภาพรวม

| คุณสมบัติ        | ค่า             |
| ---------------- | --------------- |
| **รหัส ISO**     | `th`            |
| **ชื่อท้องถิ่น** | ภาษาไทย         |
| **ระบบการเขียน** | อักษรไทย        |
| **ทิศทาง**       | ซ้ายไปขวา (LTR) |
| **ภูมิภาคหลัก**  | ประเทศไทย       |

---

## 1. ระดับความเป็นทางการ

### สำหรับ Synoptic

**ใช้ภาษาสุภาพ** — มาตรฐานสำหรับแอปพลิเคชันมืออาชีพ

| บริบท      | รูปแบบ             | ตัวอย่าง                             |
| ---------- | ------------------ | ------------------------------------ |
| UI ทั่วไป  | **คุณ + ค่ะ/ครับ** | «บันทึกโปรเจกต์ของคุณ»               |
| ข้อผิดพลาด | **สุภาพ**          | «กรุณาตรวจสอบการเชื่อมต่อของคุณ»     |
| การตลาด    | **คุณ**            | «เริ่มต้นการเดินทางสร้างสรรค์ของคุณ» |

### ระบบสรรพนาม

| สรรพนาม      | ระดับ           | ใช้ใน UI         |
| ------------ | --------------- | ---------------- |
| **ผม/ดิฉัน** | สุภาพ           | ✅ บุคคลที่หนึ่ง |
| **คุณ**      | สุภาพ, เป็นกลาง | ✅ **แนะนำ**     |
| **ท่าน**     | ทางการมาก       | สำหรับกฎหมาย     |

---

## 2. ภาษาที่เป็นกลางทางเพศ

ภาษาไทย**ไม่มีเพศทางไวยากรณ์** — เป็นกลางโดยธรรมชาติ!

```
✅ «ผู้ใช้สามารถบันทึกโปรเจกต์ได้»
✅ «นักเขียนและบรรณาธิการ»
```

### คำลงท้าย

- **ค่ะ** — สำหรับผู้หญิง
- **ครับ** — สำหรับผู้ชาย
- ใน UI ทางการ สามารถละคำลงท้ายได้

---

## 3. การเขียนภาษาไทย

### ไม่มีช่องว่างระหว่างคำ

ภาษาไทยเขียนโดย**ไม่มีช่องว่างระหว่างคำ**:

```
✅ «บันทึกโปรเจกต์ของคุณ»
❌ «บันทึก โปรเจกต์ ของ คุณ»
```

### วรรณยุกต์

มี 4 วรรณยุกต์: ไม้เอก (่), ไม้โท (้), ไม้ตรี (๊), ไม้จัตวา (๋)

## 4. คำศัพท์เทคนิค

| อังกฤษ   | ไทย        |
| -------- | ---------- |
| Save     | บันทึก     |
| Delete   | ลบ         |
| Cancel   | ยกเลิก     |
| Settings | การตั้งค่า |
| Upload   | อัปโหลด    |
| Download | ดาวน์โหลด  |

### การแปล «Ultimate»

```
✅ «สตูดิโอสองภาษาที่ดีที่สุด»
✅ «สตูดิโอสองภาษาขั้นสุดยอด»
❌ «สตูดิโอสองภาษาสุดท้าย»
```

## 5. รูปแบบตัวเลข

| ประเภท   | รูปแบบ                       | ตัวอย่าง   |
| -------- | ---------------------------- | ---------- |
| หลักพัน  | จุลภาค                       | 1,000      |
| ทศนิยม   | จุด                          | 3.14       |
| สกุลเงิน | ฿X หรือ X บาท                | ฿399       |
| วันที่   | DD/MM/YYYY หรือ วัน/เดือน/ปี | 30/12/2568 |

### ปีพุทธศักราช

ประเทศไทยใช้ปี พ.ศ. (ปีคริสต์ศักราช + 543):

- 2025 = พ.ศ. 2568
