## 1. De Usu Linguae Latinae

### Contextus Synoptic

Latina adhibetur praecipue ad:

- Textus classicos legendi et traducendi
- Opera academica et ecclesiastica
- Demonstrationes linguisticas

### Principia fundamentalia

```
✅ «Serva opus tuum» (directum, activum)
❌ «Opus tuum servabitur» (passivum)
```

### «Ultimate» Translatio

In lingua Anglica, «ultimate» significat **«optimus, summus»**.

```
✅ «Studium bilingue ultimum» — perfecta translatio
✅ «Optimum studium bilingue»
❌ «Postremum studium bilingue» (finem significat)
```

## 3. Grammatica

### Casus Latini

Lingua Latina sex casus habet:

| Casus       | Quaestio    | Exemplum |
| ----------- | ----------- | -------- |
| Nominativus | Quis? Quid? | opus     |
| Genitivus   | Cuius?      | operis   |
| Dativus     | Cui?        | operi    |
| Accusativus | Quem? Quid? | opus     |
| Ablativus   | Quo? A quo? | opere    |
| Vocativus   | —           | opus!    |

### Declinationes

Quinque declinationes nominum:

1. -a, -ae (puella)
2. -us, -i (dominus)
3. Varia (rex, corpus)
4. -us, -ūs (manus)
5. -es, -ei (dies)

### Coniugationes

Quattuor coniugationes verborum:

1. -āre (amāre)
2. -ēre (monēre)
3. -ere (legere)
4. -īre (audīre)

---

## 4. Syntaxis

### Ordo Verborum

Latina ordinem verborum liberum habet (propter casus), sed typice **SOV**:

```
Anglice: I save the document (SVO)
Latine: Ego documentum servo (SOV)
```

### Subiectum Omissibile

Latine subiectum saepe omittitur:

```
✅ «Documentum serva» (tu implicite)
```

## 8. Adnotationes Speciales

### Latina Classica vs. Ecclesiastica

- **Classica**: Cicero, Caesar — purior forma
- **Ecclesiastica**: Vulgata, documenta pontificia — pronuntiatio diversa

### Pro Synoptic

Latina adhibetur praecipue pro:

- Textibus classicis legendis
- Studiis philologicis
- Demonstrationibus linguisticis
