## 1. Ton ve Kayıt

### Synoptic Studio için

- **Kayıt**: Profesyonel ama erişilebilir
- **Ton**: Kendinden emin, net ve yardımsever — asla küçümseyici değil
- **Ses**: Aktif tercih edilir; edilgen sadece eylemi vurgulamak için

### Temel ilkeler

```
✅ «Projenizi kaydedin» (doğrudan, aktif)
❌ «Projeniz kaydedilecektir» (edilgen)

✅ «Güzel iki dilli kitaplar oluşturun»
```

---

## 2. Sen vs. Siz

### Synoptic Kuralı

**«Siz» kullanın** — profesyonel uygulamalar için standart.

| Bağlam                 | Form    | Örnek                              |
| ---------------------- | ------- | ---------------------------------- |
| Kullanıcı arayüzü      | **Siz** | «Projenizi kaydedin»               |
| Teknik dokümantasyon   | **Siz** | «PDF olarak dışa aktarabilirsiniz» |
| Hata mesajları         | **Siz** | «Bağlantınızı kontrol edin»        |
| Pazarlama (genç kitle) | **Sen** | «Yaratıcı yolculuğuna başla»       |

### ⚠️ Not

- «Siz» resmi ve saygılı — iş uygulamaları için ideal
- «Sen» samimi — genç kitleye yönelik pazarlamada kullanılabilir
- Aynı belgede karıştırmayın

## 3. Cinsiyetten Bağımsız Dil

### Türkçenin Avantajı

Türkçe **cinsiyet ayrımı yapmaz** — doğal olarak nötr!

| Zamir     | Kullanım                |
| --------- | ----------------------- |
| **O**     | Tüm cinsiyetler için ✅ |
| **Onlar** | Çoğul, nötr ✅          |

### Örnekler

```
✅ «Hoş geldiniz!» (nötr)
✅ «Kullanıcı projesini kaydedebilir»

✅ «Yazarlar ve editörler için»
❌ «Hoş geldin, sevgili kullanıcı»
```

## 4. Ünlü Uyumu

### Türkçe'nin En Önemli Kuralı

Ekler, kök kelimenin ünlülerine uyum sağlar:

| Ünlü Türü | Ünlüler    | Ek Örneği       |
| --------- | ---------- | --------------- |
| **Kalın** | a, ı, o, u | -da, -dan, -lar |
| **İnce**  | e, i, ö, ü | -de, -den, -ler |

### Örnekler

```
✅ «projede» (ince ünlüler)
✅ «dosyada» (kalın ünlüler)
❌ «projeda» (yanlış uyum)
```

### Ek Örnekleri

| Kelime | -lar/-ler    | -da/-de     | -dan/-den    |
| ------ | ------------ | ----------- | ------------ |
| Proje  | Proje**ler** | Proje**de** | Proje**den** |
| Dosya  | Dosya**lar** | Dosya**da** | Dosya**dan** |
| Kitap  | Kitap**lar** | Kitap**ta** | Kitap**tan** |

## 6. Dilbilgisi ve Söz Dizimi

### Cümle Yapısı

Türkçe **SOV** (Özne-Nesne-Yüklem) yapısını izler:

```
İngilizce: I save the project (SVO)
Türkçe: Ben projeyi kaydediyorum (SOV)
```

### Ek Yapısı (Agglutination)

Türkçe eklemeli bir dildir:

```
«Ev» → «Evlerimizden» (ev + ler + imiz + den)
= our houses + from = from our houses
```

### Fiil Çekimi

Fiiller özneye göre çekilir:

- Ben kayded**iyorum**
- Sen kayded**iyorsun**
- O kayded**iyor**
- Biz kayded**iyoruz**
- Siz kayded**iyorsunuz**
- Onlar kayded**iyorlar**

## 7. Türk Alfabesi

### Özel Harfler

Türk alfabesinde 29 harf vardır. İngilizce'de olmayan harfler:

- **ç** (ch sesi)
- **ğ** (yumuşak g, uzatma)
- **ı** (kapalı i, noktasız)
- **i** (açık i, noktalı)
- **ö** (Almanca ö gibi)
- **ş** (sh sesi)
- **ü** (Almanca ü gibi)

### ⚠️ Önemli: I/İ Farkı

Türkçe'de **büyük I** ve **küçük ı** farklıdır!

- **I/ı** — noktasız (kapalı)
- **İ/i** — noktalı (açık)

```
✅ «İndir» (doğru)
❌ «Indir» (yanlış — noktasız I)
```

## 8. Tipografi

### Tırnak İşaretleri

- Türk tırnak işaretleri: **" "** veya **« »**

```
✅ "Projeniz 'Kitabım' kaydedildi"
```

### Sayılar ve Biçimlendirme

| Tür             | Biçim         | Örnek      |
| --------------- | ------------- | ---------- |
| Binlik          | Nokta         | 1.000      |
| Ondalık         | Virgül        | 3,14       |
| Para Birimi (₺) | X ₺ veya X TL | 199,99 ₺   |
| Para Birimi (€) | X €           | 12,99 €    |
| Yüzde           | %X (önde)     | %50        |
| Tarih           | GG.AA.YYYY    | 30.12.2025 |
| Saat            | SS:DD         | 14:30      |

### Yüzde İşareti

Türkçe'de yüzde işareti **sayının önünde** gelir:

```
✅ «%50 indirim»
❌ «50% indirim»
```
