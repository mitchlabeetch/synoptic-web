## 1. Ton și Registru

### Pentru Synoptic Studio

- **Registru**: Profesional dar accesibil
- **Ton**: Încrezător, clar și util — niciodată condescendent
- **Voce**: Activă preferată; pasivă doar pentru a sublinia acțiunea

### Principii cheie

```
✅ «Salvează proiectul tău» (direct, activ)
❌ «Proiectul tău va fi salvat» (pasiv)

✅ «Creează cărți bilingve frumoase»
❌ «Cărțile bilingve pot fi create de tine»
```

---

## 2. Tu vs. Dumneavoastră

### Regula pentru Synoptic

**Folosește «tu»** — standard pentru aplicațiile moderne de tehnologie.

| Context              | Formă             | Exemplu                   |
| -------------------- | ----------------- | ------------------------- |
| Interfață utilizator | **Tu**            | «Salvează proiectul tău»  |
| Documentație tehnică | **Tu**            | «Poți exporta în PDF»     |
| Mesaje de eroare     | **Tu**            | «Verifică conexiunea ta»  |
| Texte juridice       | **Dumneavoastră** | «Utilizatorul acceptă...» |

### ⚠️ Notă

- «Tu» este informal dar prietenos — perfect pentru tech
- «Dumneavoastră» este foarte formal — doar pentru legal/oficial
- Nu amesteca formele în același document

---

## 3. Limbaj Neutru din punct de vedere al Genului

### Provocarea în română

Româna are gen gramatical (masculin, feminin, neutru). Substantivele, adjectivele și verbele se acordă.

### Strategii recomandate

| Metodă               | Exemplu                             | Utilizare        |
| -------------------- | ----------------------------------- | ---------------- |
| **Masculin generic** | «Utilizatorii»                      | Tradițional      |
| **Forme duble**      | «Utilizatori și utilizatoare»       | Formal, inclusiv |
| **Reformulare**      | «Persoanele care utilizează»        | ✅ Preferat      |
| **Pluralul**         | «Echipa ta» în loc de «Colegii tăi» | Natural          |

### Exemple practice

```
✅ «Bine ai venit!» (neutru prin context)
✅ «Proiectul a fost salvat cu succes» (pasiv — neutru)
❌ «Bine ai venit, dragă utilizator»

✅ «Pentru creatori și editori»
✅ «Pentru persoanele care publică»
```

### Capcana «Ultimate»

În engleză, «ultimate» înseamnă **«cel mai bun, suprem»**.

```
✅ «Studioul bilingv suprem»
✅ «Cel mai bun studio bilingv»
❌ «Ultimul studio bilingv» (implică sfârșitul)
```

## 5. Gramatică și Sintaxă

### Structura propoziției

Româna urmează în general **SVO** (Subiect-Verb-Obiect):

```
Engleză: I save the project (SVO)
Română: Eu salvez proiectul (SVO)
```

### Articole

Româna are articole hotărâte ENCLITICE (atașate la sfârșitul cuvântului):

| Forma nearticulată | Forma articulată |
| ------------------ | ---------------- |
| proiect            | proiect**ul**    |
| carte              | carte**a**       |
| fișier             | fișier**ul**     |

### Diacritice obligatorii

Folosește întotdeauna diacriticele corecte:

- **ă, â, î, ș, ț**

```
✅ «Șterge fișierul»
❌ «Sterge fisierul» (incorect)
```

## 6. Tipografie

### Ghilimele

- Ghilimele românești: **„ "** (jos-sus)
- Alternativă: **« »** (franceză)

```
✅ „Proiectul tău «Cartea mea» a fost salvat"
```

### Numere și formate

| Tip          | Format        | Exemplu    |

## 10. Considerații Culturale

### Formalitate

- Utilizatorii români apreciază claritatea
- Evită formalitatea excesivă (sună artificial)
- «Tu» este perfect acceptabil în contexte tech

### Anglicisme

Româna acceptă multe anglicisme în domeniul tech:

- ✅ «Cloud», «download», «upload»
- Dar preferă echivalente românești când sunt naturale:
  - ✅ «Fișier» în loc de «file»
  - ✅ «Descarcă» în loc de «downloadează»
