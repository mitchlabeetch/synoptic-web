## 1. Tone og Register

### For Synoptic Studio

- **Register**: Professionelt men tilgængeligt
- **Tone**: Selvsikker, klar og hjælpsom — aldrig nedladende
- **Stemme**: Aktiv foretrækkes; passiv kun for at understrege handlingen

### Grundprincipper

```
✅ «Gem dit projekt» (direkte, aktivt)
❌ «Dit projekt vil blive gemt» (passivt)

✅ «Skab smukke tosproget bøger»
```

---

## 2. Du vs. De

### Regel for Synoptic

**Brug «du»** — standard i moderne dansk kommunikation.

| Kontekst              | Form   | Eksempel                    |
| --------------------- | ------ | --------------------------- |
| Brugergrænseflade     | **Du** | «Gem dit projekt»           |
| Teknisk dokumentation | **Du** | «Du kan eksportere til PDF» |
| Fejlmeddelelser       | **Du** | «Tjek din forbindelse»      |
| Juridiske tekster     | **De** | «Brugeren accepterer...»    |

### ⚠️ Historisk kontekst

- «De» var tidligere den formelle form
- I dag bruges «du» i næsten alle sammenhænge
- «De» kan virke stift eller gammeldags
- Undtagelse: Meget formelle juridiske dokumenter

## 3. Kønsneutralt Sprog

### Anbefalede strategier

| Metode            | Eksempel                    | Anvendelse   |
| ----------------- | --------------------------- | ------------ |
| **De/deres**      | «Brugeren og deres projekt» | ✅ Anbefalet |
| **Vedkommende**   | «Vedkommende kan gemme»     | Formelt      |
| **Omformulering** | «Den, der bruger»           | ✅ Elegant   |
| **Flertal**       | «Brugere kan gemme»         | ✅ Naturligt |

### Eksempler

```
✅ «Velkommen!» (neutralt)
✅ «Projektet er blevet gemt» (passiv — neutralt)
❌ «Velkommen, kære bruger»

✅ «For forfattere og redaktører»
```

## 4. Oversættelsesfælder

### Falske venner

| Engelsk    | ❌ Forkert oversættelse | ✅ Korrekt oversættelse |
| ---------- | ----------------------- | ----------------------- |
| Actually   | Aktuelt                 | Faktisk, egentlig       |
| Eventually | Eventuelt               | Til sidst, endelig      |
| Fabric     | Fabrik                  | Stof                    |
| Genial     | Genial                  | Smart (genial = venlig) |

### «Ultimate»-fælden

På engelsk betyder «ultimate» ofte **«bedste, ultimative»**.

```
✅ «Det ultimative tosprogede studie» — korrekt betydning
✅ «Det bedste tosprogede studie»
❌ «Det sidste tosprogede studie» (antyder slutningen)
```

## 5. Grammatik og Syntaks

### Sætningsstruktur

Dansk følger generelt **SVO** (Subjekt-Verbum-Objekt):

```
Engelsk: I save the project (SVO)
Dansk: Jeg gemmer projektet (SVO)
```

### V2-reglen

I hovedsætninger står verbet altid på anden plads:

```
✅ «Nu kan du gemme projektet» (Adverbium-Verbum-Subjekt-Objekt)
```

### Bestemt og ubestemt form

| Ubestemt   | Bestemt       | Betydning               |
| ---------- | ------------- | ----------------------- |
| en fil     | fil**en**     | a file / the file       |
| et projekt | projekt**et** | a project / the project |

## 6. Typografi

### Citationstegn

- Danske citationstegn: **» «** eller **„ "**
- Almindelige: **" "**

```
✅ »Dit projekt 'Min bog' er gemt«
```

### Tal og formater

| Type         | Format     | Eksempel   |
| ------------ | ---------- | ---------- |
| Tusindtals   | Punktum    | 1.000      |
| Decimaler    | Komma      | 3,14       |
| Valuta (DKK) | X kr.      | 99 kr.     |
| Valuta (€)   | X €        | 12,99 €    |
| Procent      | X %        | 50 %       |
| Dato         | DD.MM.ÅÅÅÅ | 30.12.2025 |
| Tid          | HH:MM      | 14:30      |

### Særlige bogstaver

Dansk har tre ekstra bogstaver:

- **Æ, Ø, Å** — placeres sidst i alfabetet
