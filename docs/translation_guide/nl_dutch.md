## 1. Toon en Register

### Voor Synoptic Studio

- **Register**: Professioneel maar toegankelijk
- **Toon**: Zelfverzekerd, helder en behulpzaam — nooit neerbuigend
- **Stem**: Actief heeft de voorkeur; passief alleen om nadruk te leggen op de actie

### Kernprincipes

```
✅ «Sla je project op» (direct, actief)
❌ «Je project wordt opgeslagen» (passief)

✅ «Maak prachtige tweetalige boeken»
❌ «Tweetalige boeken kunnen door jou worden gemaakt»
```

## 2. Je/Jij vs. U

### Regel voor Synoptic

**Gebruik «je/jij»** — dit is de standaard voor moderne tech-applicaties.

| Context                 | Vorm   | Voorbeeld                              |
| ----------------------- | ------ | -------------------------------------- |
| Gebruikersinterface     | **Je** | «Sla je project op»                    |
| Technische documentatie | **Je** | «Je kunt exporteren naar PDF»          |
| Foutmeldingen           | **Je** | «Controleer je verbinding»             |
| Juridische teksten      | **U**  | «Door akkoord te gaan, bevestigt u...» |

### Belgisch Nederlands (Vlaams)

In Vlaanderen is **«u»** gangbaarder in formele contexten, maar voor tech-toepassingen is **«je»** ook acceptabel.

```
✅ Nederland: « Je project is opgeslagen »
✅ België formeel: « Uw project is opgeslagen »
✅ België informeel: « Je project is opgeslagen » (ook OK)
```

## 3. Genderneutraal Taalgebruik

### Aanbevolen opties

| Methode            | Voorbeeld                  | Gebruik       |
| ------------------ | -------------------------- | ------------- |
| **Zij/hen**        | gebruiker en hun project   | ✅ Aanbevolen |
| **Die/diens**      | de auteur en diens werk    | Formeler      |
| **Herformulering** | wie gebruikt, personen die | ✅ Voorkeur   |
| **Meervoud**       | Gebruikers kunnen hun...   | ✅ Natuurlijk |

### Voor Synoptic

```
✅ « Welkom! »
❌ « Welkom, beste gebruiker »

✅ « Schrijvers en uitgevers »
✅ « Mensen die publiceren »
❌ « De auteurs » (kan, maar liever neutraler)

✅ « Je team »
❌ « Je collega's » (kan mannelijk impliceren)
```

### Beroepsnamen

Nederlandse vormen zijn vaak al neutraal of hebben vrouwelijke varianten:

| Mannelijk | Vrouwelijk  | Neutraal/Inclusief       |
| --------- | ----------- | ------------------------ |
| Auteur    | Auteur      | Auteur ✅                |
| Schrijver | Schrijfster | Schrijver / Wie schrijft |
| Vertaler  | Vertaalster | Vertaler / Vertalende    |

## 4. Valse Vrienden en Valkuilen

### Valse vrienden Engels-Nederlands

| Engels      | ❌ Verkeerde vertaling | ✅ Juiste vertaling  |
| ----------- | ---------------------- | -------------------- |
| Eventually  | Eventueel              | Uiteindelijk         |
| Fabric      | Fabriek                | Stof                 |
| Actually    | Actueel                | Eigenlijk, in feite  |
| Bless       | Blessen                | Zegenen              |
| Novel       | Novelle                | Roman                |
| Sympathetic | Sympathiek             | Meelevend, begripvol |

### De «Ultimate»-valkuil

In het Engels betekent «ultimate» vaak **«de beste, de ultieme»**.  
In het Nederlands betekent «ultiem» ook **«de beste»**, maar kan ook «laatste» impliceren.

```
✅ « De ultieme tweetalige studio » (correct in Nederlandse context)
✅ « De beste tweetalige studio »
```

## 5. Grammatica en Zinsbouw

### Zinsstructuur

- **Onderwerp-Werkwoord-Voorwerp (OWV)** is standaard
- Let op de **werkwoordspositie** in bijzinnen: werkwoord gaat naar het einde

```
✅ Hoofdzin: « Je kunt je project exporteren »
✅ Bijzin: « ...zodat je je project kunt exporteren »
```

### Lidwoorden

- **De** (mannelijk/vrouwelijk): de studio, de pagina
- **Het** (onzijdig): het project, het bestand

### Samengestelde woorden

Nederlands voegt woorden samen **zonder spaties**:

```
✅ Projectinstellingen
❌ Project instellingen
❌ Project-instellingen (alleen bij potentiële verwarring)

✅ PDF-export (afkorting + woord)
✅ E-mail
```

## 6. Typografie

### Aanhalingstekens

- Nederlandse aanhalingstekens: **„ "** of **" "**
- Enkele aanhalingstekens voor citaten binnen citaten: **' '**

```
✅ „Je project 'Mijn boek' is opgeslagen"
✅ "Je project 'Mijn boek' is opgeslagen"
```

### Getallen en formaten

| Type          | Formaat        | Voorbeeld      |
| ------------- | -------------- | -------------- |
| Duizendtallen | Punt           | 1.000          |
| Decimalen     | Komma          | 3,14           |
| Valuta        | € X,XX         | € 12,99        |
| Percentage    | X%             | 50%            |
| Datum         | DD-MM-JJJJ     | 30-12-2025     |
| Tijd          | HH:MM of HH.MM | 14:30 of 14.30 |

### Opmerking over valuta

Het euroteken staat **vóór** het bedrag in Nederland: **€ 12,99**

## 8. Interfacepatronen

### Knoppen

- Gebruik het **infinitief** of **gebiedende wijs**
- Maximaal 1-3 woorden

| Actie    | Nederlands    |
| -------- | ------------- |
| Save     | Opslaan       |
| Export   | Exporteren    |
| Delete   | Verwijderen   |
| Cancel   | Annuleren     |
| Create   | Maken / Nieuw |
| Edit     | Bewerken      |
| Sign in  | Inloggen      |
| Sign out | Uitloggen     |

### Foutmeldingen

- Wees **specifiek** over het probleem
- Bied een **oplossing**
- Geef de gebruiker niet de schuld

```
✅ « Verbinding verbroken. Controleer je internetverbinding en probeer het opnieuw. »
❌ « Netwerkfout »
❌ « Je hebt de verbinding verloren »
```

## 10. Culturele Overwegingen

### Directheid

Nederlandse communicatie is vrij **direct** — dit wordt niet als onbeleefd gezien. In UI:

- Wees duidelijk en to-the-point
- Vermijd overdreven beleefdheidsfrasen

### Anglicismen

Nederlands accepteert veel anglicismen in tech:

- ✅ « Uploaden », « downloaden », « file », « cloud »
- Maar Nederlandse alternatieven zijn vaak beter:
  - ✅ « Bestand » i.p.v. « file »
  - ✅ « Koppeling » i.p.v. « link » (contextafhankelijk)
