## 1. To i Registre

### Per a Synoptic Studio

- **Registre**: Professional però accessible
- **To**: Segur, clar i útil — mai condescendent
- **Veu**: Activa preferida; passiva només per emfatitzar l'acció

### Principis clau

```
✅ «Desa el teu projecte» (directe, actiu)
❌ «El teu projecte serà desat» (passiu)

✅ «Crea llibres bilingües preciosos»
```

---

## 2. Tu vs. Vostè

### Regla per a Synoptic

**Utilitza «tu»** — estàndard per a aplicacions tecnològiques modernes.

| Context              | Forma     | Exemple                     |
| -------------------- | --------- | --------------------------- |
| Interfície d'usuari  | **Tu**    | «Desa el teu projecte»      |
| Documentació tècnica | **Tu**    | «Pots exportar a PDF»       |
| Missatges d'error    | **Tu**    | «Comprova la teva connexió» |
| Textos legals        | **Vostè** | «L'usuari accepta...»       |

### ⚠️ Nota

- «Tu» és informal però amigable — perfecte per a tech
- «Vostè» és molt formal — només per a legal/oficial

---

## 3. Llenguatge Inclusiu de Gènere

### Repte en català

El català té gènere gramatical (masculí, femení). Verbs i adjectius concorden.

### Estratègies recomanades

| Mètode             | Exemple                 | Ús           |
| ------------------ | ----------------------- | ------------ |
| **Desdoblament**   | «Usuaris i usuàries»    | Formal       |
| **Formes neutres** | «Les persones usuàries» | ✅ Preferit  |
| **Barra**          | «Usuari/ària»           | Evitar en UI |
| **Col·lectius**    | «L'equip»               | Natural      |

### Per a Synoptic

```
✅ «Benvingut/da!» o simplement «Hola!»
✅ «El projecte s'ha desat correctament» (impersonal)
❌ «Benvingut, estimat usuari»

✅ «Per a creadors i creadores»
✅ «Per a qui publica»
```

## 4. Paranys de Traducció

### Falsos amics

| Anglès     | ❌ Mala traducció | ✅ Bona traducció |
| ---------- | ----------------- | ----------------- |
| Actually   | Actualment        | En realitat       |
| Eventually | Eventualment      | Finalment         |
| Library    | Llibreria         | Biblioteca        |
| Sensible   | Sensible          | Raonable          |
| Resume     | Resum             | Reprendre         |

### El parany «Ultimate»

En anglès, «ultimate» sovint significa **«el millor, el definitiu»**.

```
✅ «L'estudi bilingüe definitiu»
✅ «El millor estudi bilingüe»
❌ «L'últim estudi bilingüe» (implica final)
```

## 5. Gramàtica i Sintaxi

### Estructura de la frase

El català segueix generalment **SVO** (Subjecte-Verb-Objecte):

```
Anglès: I save the project (SVO)
Català: Jo deso el projecte (SVO)
```

### Articles

- El/La, Els/Les (determinats)
- Un/Una, Uns/Unes (indeterminats)
- L' (elisió davant vocal)

### El punt volat (·)

Utilitzat per a la ela geminada:

- **col·laborar**, **intel·ligent**, **cancel·la**

## 6. Tipografia

### Cometes

- Cometes angulars preferides: **« »**
- Alternativa: **" "**

```
✅ «El teu projecte "El meu llibre" s'ha desat»
```

### Números i formats

| Tipus       | Format     | Exemple    |
| ----------- | ---------- | ---------- |
| Milers      | Punt       | 1.000      |
| Decimals    | Coma       | 3,14       |
| Moneda      | X €        | 12,99 €    |
| Percentatge | X %        | 50 %       |
| Data        | DD/MM/AAAA | 30/12/2025 |
| Hora        | HH:MM      | 14:30      |

### Missatges d'error

```
✅ Bon exemple:
«S'ha perdut la connexió. Comprova la connexió a internet i torna-ho a provar.»

❌ Mal exemple:
«Error de xarxa» (no específic)
«Has perdut la connexió» (acusa l'usuari)
```
