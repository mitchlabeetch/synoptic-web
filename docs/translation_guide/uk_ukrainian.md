## 1. Тон і регістр

### Для Synoptic Studio

- **Регістр**: Професійний, але доступний
- **Тон**: Впевнений, чіткий і корисний — ніколи зверхній
- **Голос**: Активний стан переважно; пасивний лише для акценту на дії

### Ключові принципи

```
✅ «Збережи свій проєкт» (прямий, активний)
❌ «Твій проєкт буде збережено» (пасивний)

✅ «Створюй прекрасні двомовні книги»
```

---

## 2. Ти vs. Ви

### Правило для Synoptic

**Використовуйте «ти»** — стандарт для сучасних технологічних застосунків.

| Контекст                 | Форма  | Приклад                       |
| ------------------------ | ------ | ----------------------------- |
| Інтерфейс користувача    | **Ти** | «Збережи свій проєкт»         |
| Технічна документація    | **Ти** | «Ти можеш експортувати в PDF» |
| Повідомлення про помилки | **Ти** | «Перевір своє з'єднання»      |
| Юридичні тексти          | **Ви** | «Користувач погоджується...»  |

### Написання займенників

- **ти/твій** — з малої літери в інтерфейсі
- **Ви/Ваш** — з великої літери при ввічливому зверненні до однієї особи

## 3. Гендерно-нейтральна мова

### Виклики в українській

Українська має граматичний рід (чоловічий, жіночий, середній). Дієслова та прикметники узгоджуються.

### Рекомендовані стратегії

| Метод                      | Приклад              | Застосування          |
| -------------------------- | -------------------- | --------------------- |
| **Безособові конструкції** | «Можна експортувати» | ✅ Переважно          |
| **Множина**                | «Користувачі можуть» | ✅ Нейтрально         |
| **Нейтральні іменники**    | «Особа», «людина»    | Для юридичних текстів |
| **Подвійні форми**         | «Автори та авторки»  | Інклюзивно            |

### Для Synoptic

```
✅ «Ласкаво просимо!» (нейтрально)
✅ «Проєкт збережено» (безособово)
❌ «Ласкаво просимо, дорогий користувачу»

✅ «Для тих, хто створює та публікує»
✅ «Для творців контенту»
```

| Accurate   | Акуратний                | Точний                 |
| Actual     | Актуальний               | Фактичний, справжній   |
| Magazine   | Магазин                  | Журнал                 |
| Prospect   | Проспект                 | Перспектива            |
| Sympathy   | Симпатія                 | Співчуття              |

### Пастка «Ultimate»

В англійській «ultimate» часто означає **«найкращий, неперевершений»**.  
В українській «ультимативний» пов'язано з **ультиматумом** (вимога).

```
✅ «Найкраща двомовна студія»
✅ «Ідеальна двомовна студія»
❌ «Ультимативна двомовна студія» (негативна конотація)
```

## 5. Граматика та синтаксис

### Структура речення

- **Підмет-Присудок-Додаток (ППД)** — стандарт, але порядок слів гнучкий
- Флексії дозволяють змінювати порядок для акценту

### Відмінки

Українська має 7 відмінків:

| Відмінок  | Питання           | Приклад      |
| --------- | ----------------- | ------------ |
| Називний  | Хто? Що?          | проєкт       |
| Родовий   | Кого? Чого?       | проєкту      |
| Давальний | Кому? Чому?       | проєкту      |
| Знахідний | Кого? Що?         | проєкт       |
| Орудний   | Ким? Чим?         | проєктом     |
| Місцевий  | На кому? На чому? | (на) проєкті |
| Кличний   | —                 | проєкте!     |

## 6. Типографіка

### Лапки

- Українські лапки: **« »** (ялинки) — основні
- Вкладені: **„ "** (лапки)

```
✅ «Твій проєкт „Моя книга" збережено»
```

### Числа та формати

| Тип        | Формат        | Приклад    |
| ---------- | ------------- | ---------- |
| Тисячі     | Пробіл        | 1 000      |
| Десяткові  | Кома          | 3,14       |
| Валюта (₴) | X ₴ або X грн | 499 ₴      |
| Валюта (€) | X €           | 12,99 €    |
| Відсоток   | X%            | 50%        |
| Дата       | ДД.ММ.РРРР    | 30.12.2025 |
| Час        | ГГ:ХХ         | 14:30      |

### Буква Ґ та Апостроф

- Використовуйте **«ґ»** де потрібно: ґудзик, ґанок
- Апостроф замість російського м'якого знака: **з'єднання**, **пам'ять**
