## 1. Sävy ja Rekisteri

### Synoptic Studiolle

- **Rekisteri**: Ammattimainen mutta helposti lähestyttävä
- **Sävy**: Itsevarma, selkeä ja avulias — ei koskaan alentava
- **Ääni**: Aktiivi suositeltava; passiivi vain toiminnan korostamiseen

### Perusperiaatteet

```
✅ «Tallenna projektisi» (suora, aktiivi)
❌ «Projektisi tallennetaan» (passiivi)

✅ «Luo kauniita kaksikielisiä kirjoja»
```

---

## 2. Sinuttelu vs. Teitittely

### Synopticin sääntö

**Käytä sinuttelua** — moderni standardi Suomessa.

| Konteksti              | Muoto    | Esimerkki                |
| ---------------------- | -------- | ------------------------ |
| Käyttöliittymä         | **Sinä** | «Tallenna projektisi»    |
| Tekninen dokumentaatio | **Sinä** | «Voit viedä PDF-muotoon» |
| Virheilmoitukset       | **Sinä** | «Tarkista yhteytesi»     |
| Oikeudellinen          | **Te**   | «Käyttäjä hyväksyy...»   |

### ⚠️ Huomio

- Sinuttelu on yleistynyt ja hyväksyttävää
- Teitittely (te) vain erityisen muodollisissa tilanteissa
- Nuorempi sukupolvi odottaa sinuttelua

---

## 3. Sukupuolineutraali Kieli

### Suomen kielen etu

Suomessa **ei ole kieliopillista sukua** — luonnostaan neutraali!

| Pronomini | Käyttö                   |
| --------- | ------------------------ |
| **Hän**   | Kaikille sukupuolille ✅ |
| **He**    | Monikko, neutraali ✅    |

### Esimerkkejä

```
✅ «Tervetuloa!» (neutraali)
✅ «Käyttäjä voi tallentaa projektinsa»

✅ «Kirjoittajille ja toimittajille»
❌ «Tervetuloa, hyvä käyttäjä»
```

## 4. Sijamuodot

### Suomen 15 sijamuotoa

Suomi on agglutinoiva kieli, jossa on 15 sijamuotoa. Tärkeimmät:

| Sija        | Pääte     | Esimerkki       | Käyttö       |
| ----------- | --------- | --------------- | ------------ |
| Nominatiivi | —         | projekti        | Subjekti     |
| Genetiivi   | -n        | projekti**n**   | Omistus      |
| Partitiivi  | -a/-ä     | projekti**a**   | Osittaisuus  |
| Akkusatiivi | -(n)      | projekti(n)     | Objekti      |
| Inessiivi   | -ssa/-ssä | projekti**ssa** | Sisällä      |
| Elatiivi    | -sta/-stä | projekti**sta** | Sisältä ulos |
| Illatiivi   | -Vn/-hVn  | projekti**in**  | Sisälle      |
| Adessiivi   | -lla/-llä | projekti**lla** | Päällä       |
| Ablatiivi   | -lta/-ltä | projekti**lta** | Päältä pois  |
| Allatiivi   | -lle      | projekti**lle** | Päälle       |

### Esimerkki käytännössä

```
«Tallenna projekti» → «Tallenna projekti**n**» (genetiivi/akkusatiivi)
«Vie projekti PDF-muotoon» → «Vie projekti PDF-muoto**on**» (illatiivi)
```

## 5. Käännössudenkuopat

### Väärät ystävät

| Englanti    | ❌ Väärä käännös | ✅ Oikea käännös |
| ----------- | ---------------- | ---------------- |
| Actual      | Aktuaalinen      | Todellinen       |
| Eventually  | Mahdollisesti    | Lopulta          |
| Sympathetic | Sympaattinen     | Myötätuntoinen   |

### «Ultimate»-ansa

Englanniksi «ultimate» tarkoittaa usein **«paras, lopullinen»**.

```
✅ «Lopullinen kaksikielinen studio»
✅ «Paras kaksikielinen studio»
❌ «Viimeinen kaksikielinen studio» (viittaa loppumiseen)
```

## 6. Typografia

### Lainausmerkit

- Suomalaiset lainausmerkit: **" "** tai **» »**
- Yksinkertaiset: **' '**

```
✅ "Projektisi 'Kirjani' on tallennettu"
```

### Numerot ja muodot

| Tyyppi       | Muoto      | Esimerkki  |
| ------------ | ---------- | ---------- |
| Tuhannet     | Välilyönti | 1 000      |
| Desimaalit   | Pilkku     | 3,14       |
| Valuutta (€) | X €        | 12,99 €    |
| Prosentti    | X %        | 50 %       |
| Päivämäärä   | PP.KK.VVVV | 30.12.2025 |
| Aika         | HH:MM      | 14:30      |

### Erityismerkit

Suomessa on kaksi lisäkirjainta:

- **Ä, Ö** — sijoitetaan aakkoston loppuun
