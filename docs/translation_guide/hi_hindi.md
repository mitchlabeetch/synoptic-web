## 1. टोन और रजिस्टर

### Synoptic Studio के लिए

- **रजिस्टर**: सम्मानजनक लेकिन सुलभ
- **टोन**: आत्मविश्वासी, स्पष्ट और सहायक — कभी भी अहंकारी नहीं
- **शैली**: सक्रिय वाक्य प्रमुख; कर्मवाच्य केवल क्रिया पर जोर देने के लिए

### मुख्य सिद्धांत

```
✅ «अपना प्रोजेक्ट सेव करें» (प्रत्यक्ष, सक्रिय)
❌ «आपका प्रोजेक्ट सेव किया जाएगा» (निष्क्रिय)

✅ «सुंदर द्विभाषी पुस्तकें बनाएं»
❌ «द्विभाषी पुस्तकें आपके द्वारा बनाई जा सकती हैं»
```

## 2. औपचारिकता स्तर: तुम vs. आप

### Synoptic का नियम

**«आप» का उपयोग करें** — पेशेवर ऐप्स के लिए मानक।

| संदर्भ       | रूप        | उदाहरण                           |
| ------------ | ---------- | -------------------------------- |
| UI सामान्य   | **आप**     | «अपना प्रोजेक्ट सेव करें»        |
| त्रुटि संदेश | **आप**     | «कृपया अपना कनेक्शन जांचें»      |
| मार्केटिंग   | **आप/तुम** | «अपनी रचनात्मक यात्रा शुरू करें» |
| कानूनी       | **आप**     | «उपयोगकर्ता सहमत होता है...»     |

### सर्वनाम पदानुक्रम

| सर्वनाम | औपचारिकता      | UI में उपयोग    |
| ------- | -------------- | --------------- |
| **तू**  | बहुत अनौपचारिक | ❌ उपयोग न करें |
| **तुम** | अनौपचारिक      | △ सीमित उपयोग   |
| **आप**  | सम्मानजनक      | ✅ **अनुशंसित** |

### ⚠️ ध्यान दें

- «आप» एकवचन और बहुवचन दोनों के लिए काम करता है
- क्रिया रूप «आप» के साथ बहुवचन होते हैं: «आप करते हैं»

## 3. लिंग-तटस्थ भाषा

### हिंदी की चुनौती

हिंदी में व्याकरणिक लिंग है (पुल्लिंग/स्त्रीलिंग)। क्रियाएं और विशेषण लिंग के अनुसार बदलते हैं।

### अनुशंसित रणनीतियाँ

| विधि                | उदाहरण                    | उपयोग                   |
| ------------------- | ------------------------- | ----------------------- |
| **पुल्लिंग बहुवचन** | «उपयोगकर्ता»              | पारंपरिक रूप से समावेशी |
| **तटस्थ संज्ञाएं**  | «व्यक्ति», «लोग»          | ✅ प्राथमिक             |
| **कर्मवाच्य**       | «प्रोजेक्ट सेव किया गया»  | ✅ UI के लिए अच्छा      |
| **दोहरा रूप**       | «उपयोगकर्ता/उपयोगकर्ताएं» | लंबा लेकिन स्पष्ट       |

### व्यावहारिक उदाहरण

```
✅ «आपका स्वागत है!» (तटस्थ)
✅ «प्रोजेक्ट सफलतापूर्वक सेव हो गया» (कर्मवाच्य)
❌ «प्रिय उपयोगकर्ता» (केवल पुल्लिंग)

✅ «रचनाकारों के लिए»
✅ «सभी लेखकों के लिए»
```

## 4. अनुवाद जाल और गलतियाँ

### झूठे मित्र (False Friends)

| अंग्रेज़ी  | ❌ गलत अनुवाद | ✅ सही अनुवाद    | कारण                    |
| ---------- | ------------- | ---------------- | ----------------------- |
| Ultimate   | अंतिम         | परम/सर्वश्रेष्ठ  | «ultimate» = सबसे अच्छा |
| Save       | बचाओ          | सेव करें/सहेजें  | संदर्भ अलग              |
| Actually   | वास्तव में    | असल में          | —                       |
| Eventually | अंततः         | अंत में          | —                       |
| Resume     | रिज्यूमे      | फिर से शुरू करें | संदर्भ पर निर्भर        |

### «Ultimate» का अनुवाद

अंग्रेज़ी में «ultimate» का अर्थ है **«सर्वश्रेष्ठ, परम»**।

```
✅ «सर्वश्रेष्ठ द्विभाषी स्टूडियो»
✅ «परम द्विभाषी स्टूडियो»
❌ «अंतिम द्विभाषी स्टूडियो» (समाप्ति का अर्थ)
```

## 5. व्याकरण और वाक्य-विन्यास

### वाक्य संरचना

हिंदी **SOV** (कर्ता-कर्म-क्रिया) संरचना का पालन करती है:

```
अंग्रेज़ी: I save the project (SVO)
हिंदी: मैं प्रोजेक्ट सेव करता हूं (SOV)
```

### परसर्ग (Postpositions)

हिंदी में परसर्ग का उपयोग होता है (अंग्रेज़ी के prepositions के विपरीत):

| परसर्ग | कार्य     | उदाहरण                      |
| ------ | --------- | --------------------------- |
| को     | को/के लिए | «प्रोजेक्ट **को** सेव करें» |
| में    | में       | «फ़ोल्डर **में** सेव करें»  |
| से     | से        | «क्लाउड **से** डाउनलोड»     |
| पर     | पर        | «बटन **पर** क्लिक करें»     |
| के लिए | के लिए    | «आप **के लिए** बनाया गया»   |

### क्रिया संयुग्मन

क्रियाएं लिंग, वचन, काल और पक्ष के अनुसार बदलती हैं:

```
✅ «फ़ाइल सेव हो गई» (स्त्रीलिंग)
✅ «प्रोजेक्ट सेव हो गया» (पुल्लिंग)
```

## 6. देवनागरी लिपि

### विशेषताएं

- स्वर चिह्न (मात्राएं) व्यंजनों से जुड़ते हैं
- शिरोरेखा (headline) अक्षरों को जोड़ती है
- बाएं से दाएं लिखी जाती है

### विराम चिह्न

| चिह्न         | हिंदी      | अंग्रेज़ी |
| ------------- | ---------- | --------- |
| पूर्ण विराम   | ।          | .         |
| अल्पविराम     | ,          | ,         |
| प्रश्न चिह्न  | ?          | ?         |
| विस्मयादिबोधक | !          | !         |
| उद्धरण चिह्न  | " " या « » | ""        |

### संख्या और प्रारूप

| प्रकार     | प्रारूप    | उदाहरण     |
| ---------- | ---------- | ---------- |
| हज़ार      | कॉमा       | 1,000      |
| दशमलव      | बिंदु      | 3.14       |
| मुद्रा (₹) | ₹X         | ₹999       |
| मुद्रा (€) | €X         | €12.99     |
| प्रतिशत    | X%         | 50%        |
| तारीख      | DD/MM/YYYY | 30/12/2025 |
| समय        | HH:MM      | 14:30      |

### भारतीय अंक प्रणाली

भारत में लाख और करोड़ का उपयोग:

- 1,00,000 = एक लाख
- 1,00,00,000 = एक करोड़
