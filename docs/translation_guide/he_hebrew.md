## 1. כיוון טקסט (RTL)

### כללים בסיסיים

- טקסט עברי נכתב **מימין לשמאל**
- מספרים נקראים משמאל לימין
- ממשק המשתמש צריך להיות **מראה** (mirrored)

### השלכות על העיצוב

```
✅ כפתורים ותפריטים בצד ימין
✅ סרגל גלילה בצד שמאל
✅ חיצים מתהפכים (→ הופך ל-←)
```

### טקסט דו-כיווני (Bidirectional)

כאשר משלבים עברית עם אנגלית או מספרים:

```
✅ «הקובץ יוצא בהצלחה ל-PDF»
✅ «יש לך 5 פרויקטים חדשים»
```

## 2. רמת פנייה

### עבור Synoptic

**השתמש בפנייה ישירה (אתה/את)** — סטנדרט לאפליקציות מודרניות.

| הקשר       | צורה       | דוגמה                      |
| ---------- | ---------- | -------------------------- |
| ממשק משתמש | **אתה/את** | «שמור את הפרויקט שלך»      |
| רשמי       | **אתה/את** | «אנא בדוק את החיבור שלך»   |
| שגיאות     | **מנומס**  | «אירעה שגיאה, נסה שנית»    |
| שיווק      | **אתה/את** | «התחל את המסע היצירתי שלך» |

### ⚠️ הערה

- עברית מודרנית משתמשת בעיקר ב«אתה/את»
- צורת «אתם» לפנייה מכבדת ליחיד היא ארכאית

## 3. שפה ניטרלית מגדרית

### האתגר

עברית היא שפה עם מין דקדוקי חזק. כל פועל, שם תואר ושם עצם משתנים לפי מין.

### אסטרטגיות מומלצות

| שיטה            | דוגמה                | שימוש             |
| --------------- | -------------------- | ----------------- |
| **כתיב מלא**    | «משתמשים/ות»         | פורמלי            |
| **צורת רבים**   | «המשתמשים והמשתמשות» | רשמי              |
| **סגנון סביל**  | «הקובץ נשמר»         | ✅ **מועדף ל-UI** |
| **פנייה כללית** | «לקוחות יקרים»       | שיווק             |

### דוגמאות מעשיות

```
✅ «ברוכים הבאים!» (רבים — כולל את כולם)
✅ «הפרויקט נשמר בהצלחה» (סביל — ניטרלי)
❌ «ברוך הבא, משתמש» (זכר מפורש)

✅ «ליוצרים וליוצרות»
✅ «לאנשי יצירה» (צורה ניטרלית)
```

## 4. מלכודות תרגום

### ידידים כוזבים (False Friends)

| אנגלית     | ❌ תרגום שגוי | ✅ תרגום נכון    | הסבר                    |
| ---------- | ------------- | ---------------- | ----------------------- |
| Ultimate   | אחרון         | אולטימטיבי/מושלם | «ultimate» = הטוב ביותר |
| Save       | להציל         | לשמור            | הקשר שונה               |
| Library    | ספרייה        | ספרייה           | אותה מילה — תקין        |
| Actually   | אקטואלי       | למעשה/בעצם       | —                       |
| Eventually | אוונטואלי     | בסופו של דבר     | —                       |

### תרגום «Ultimate»

באנגלית, «ultimate» פירושו **«הטוב ביותר, האולטימטיבי»**.
בעברית, «אולטימטיבי» היא מילה מושאלת עם משמעות חיובית דומה.

```
✅ «הסטודיו הדו-לשוני האולטימטיבי»
✅ «הסטודיו הדו-לשוני המושלם»
❌ «הסטודיו הדו-לשוני האחרון» (משמע: האחרון בסדרה)
```

## 5. דקדוק ותחביר

### מבנה משפט

עברית בדרך כלל עוקבת אחר **SVO** (נושא-נשוא-מושא):

```
✅ «שמור את הפרויקט» (פועל + מושא)
✅ «המשתמש שומר את הפרויקט» (נושא + נשוא + מושא)
```

### הטיות

עברית היא שפה מורפולוגית עשירה:

- פעלים מוטים לפי מין, מספר, זמן וגוף
- שמות עצם מוטים לפי מין ומספר

### סמיכות

שימו לב לסמיכות בעברית:

```
✅ «הגדרות המשתמש» (נקוד: הַגְדָּרוֹת)
✅ «קובץ הפרויקט»
```
