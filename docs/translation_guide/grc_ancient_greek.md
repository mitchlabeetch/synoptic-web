## 1. Περὶ τῆς Χρήσεως

### Πλαίσιον Synoptic

Ἡ ἀρχαία Ἑλληνικὴ χρησιμοποιεῖται κυρίως διά:

- Κλασσικὰ κείμενα
- Φιλολογικὰς μελέτας
- Θεολογικὰ ἔργα (Καινὴ Διαθήκη)

## 3. Γραμματική

### Πτώσεις

Ἡ ἀρχαία Ἑλληνικὴ πέντε πτώσεις ἔχει:

| Πτῶσις     | Ἐρώτησις  | Παράδειγμα |
| ---------- | --------- | ---------- |
| Ὀνομαστική | Τίς; Τί;  | ἔργον      |
| Γενική     | Τίνος;    | ἔργου      |
| Δοτική     | Τίνι;     | ἔργῳ       |
| Αἰτιατική  | Τίνα; Τί; | ἔργον      |
| Κλητική    | —         | ἔργον!     |

### Ἄρθρα

- ὁ, ἡ, τό (ὁριστικόν)
- Οὐδὲν ἀόριστον ἄρθρον

### Κλίσεις

Τρεῖς κλίσεις ὀνομάτων:

1. α-κλίσις (γλῶττα)
2. ο-κλίσις (λόγος)
3. Σύμφωνα (βασιλεύς)

---

## 4. Τονισμός

### Πνεύματα καὶ Τόνοι

Ἡ ἀρχαία Ἑλληνικὴ χρησιμοποιεῖ:

- **Πνεῦμα δασύ** (῾): ἱστορία
- **Πνεῦμα ψιλόν** (᾿): ἔργον
- **Ὀξεῖα** (´): λόγος
- **Βαρεῖα** (`): τὸν λόγον
- **Περισπωμένη** (῀): γλῶττα

### ⚠️ Σημαντικόν

Πάντοτε χρησιμοποίει τοὺς τόνους — ἀλλάττουν τὴν σημασίαν!

## 5. Διαλέκτοι

| Διάλεκτος  | Χρῆσις                                |
| ---------- | ------------------------------------- |
| **Ἀττική** | Κλασσικά κείμενα (Πλάτων, Θουκυδίδης) |
| **Ἰωνική** | Ἡρόδοτος                              |
| **Κοινή**  | Καινὴ Διαθήκη ✅ Συχνοτέρα            |
| **Δωρική** | Ποίησις                               |

**Σύστασις:** Χρησιμοποίει τὴν Κοινὴν ἢ τὴν Ἀττικήν.

## 9. Σημειώσεις

### Κλασσικὴ vs Βυζαντινή

- **Κλασσική**: 5ος π.Χ. - 4ος π.Χ.
- **Κοινή**: 4ος π.Χ. - 4ος μ.Χ.
- **Βυζαντινή**: 4ος μ.Χ. - 15ος μ.Χ.

### Χρῆσις ἐν Synoptic

Ἡ ἀρχαία Ἑλληνικὴ χρησιμοποιεῖται διά:

- Κλασσικὰ κείμενα
- Φιλολογικὰς ἐργασίας
- Ἐπιδείξεις γλωσσολογικάς
