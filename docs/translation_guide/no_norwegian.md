## 1. Bokmål vs. Nynorsk

### For Synoptic

**Bruk Bokmål** — det mest brukte skriftspråket (ca. 85% av befolkningen).

| Språkform   | Bruk        | Eksempel «Save»   |
| ----------- | ----------- | ----------------- |
| **Bokmål**  | ✅ Standard | «Lagre»           |
| **Nynorsk** | Alternativ  | «Lagre» / «Lagra» |

### Forskjeller mellom Bokmål og Nynorsk

| Konsept | Bokmål  | Nynorsk |
| ------- | ------- | ------- |
| I (jeg) | Jeg     | Eg      |
| Vi      | Vi      | Me/Vi   |
| Ikke    | Ikke    | Ikkje   |
| Hva     | Hva     | Kva     |
| Hvordan | Hvordan | Korleis |

## 2. Du vs. De

### Regel for Synoptic

**Bruk «du»** — standard i moderne norsk.

| Kontekst              | Form   | Eksempel                    |
| --------------------- | ------ | --------------------------- |
| Brukergrensesnitt     | **Du** | «Lagre prosjektet ditt»     |
| Teknisk dokumentasjon | **Du** | «Du kan eksportere til PDF» |
| Feilmeldinger         | **Du** | «Sjekk tilkoblingen din»    |
| Juridiske tekster     | **De** | «Brukeren aksepterer...»    |

### ⚠️ Historisk kontekst

- «De» var den formelle formen
- I dag er «du» standard i nesten alle sammenhenger
- «De» kan virke stivt eller gammeldags

## 3. Kjønnsnøytralt Språk

### Anbefalte strategier

| Metode            | Eksempel                     | Bruk                      |
| ----------------- | ---------------------------- | ------------------------- |
| **De/deres**      | «Brukeren og deres prosjekt» | ✅ Anbefalt               |
| **Hen**           | «Hen kan lagre»              | Nyere pronomen, akseptert |
| **Vedkommende**   | «Vedkommende kan»            | Formelt                   |
| **Omformulering** | «Den som bruker»             | ✅ Elegant                |

### Pronomen «hen»

Norsk har adoptert «hen» som kjønnsnøytralt pronomen:

- **Han** = he
- **Hun** = she
- **Hen** = de (entall, kjønnsnøytralt) ✅

### Eksempler

```
✅ «Velkommen!» (nøytralt)
✅ «Prosjektet er lagret» (passiv — nøytralt)
❌ «Velkommen, kjære bruker»

✅ «For forfattere og redaktører»
```

## 4. Oversettelsesfeller

### Falske venner

| Engelsk    | ❌ Feil oversettelse | ✅ Riktig oversettelse |
| ---------- | -------------------- | ---------------------- |
| Actually   | Aktuelt              | Faktisk, egentlig      |
| Eventually | Eventuelt            | Til slutt, etter hvert |
| Pathetic   | Patetisk             | Stakkarslig            |

### «Ultimate»-fellen

På engelsk betyr «ultimate» ofte **«beste, ultimate»**.

```
✅ «Det ultimate tospråklige studioet» — riktig betydning
✅ «Det beste tospråklige studioet»
❌ «Det siste tospråklige studioet» (antyder slutten)
```

## 5. Grammatikk og Syntaks

### Setningsstruktur

Norsk følger generelt **SVO** (Subjekt-Verb-Objekt):

```
Engelsk: I save the project (SVO)
Norsk: Jeg lagrer prosjektet (SVO)
```

### V2-regelen

I hovedsetninger står verbet alltid på andre plass:

```
✅ «Nå kan du lagre prosjektet» (Adverb-Verb-Subjekt-Objekt)
```

### Bestemt og ubestemt form

| Ubestemt    | Bestemt              | Betydning               |
| ----------- | -------------------- | ----------------------- |
| en fil      | fil**en** / fil**a** | a file / the file       |
| et prosjekt | prosjekt**et**       | a project / the project |

---

## 6. Typografi

### Anførselstegn

- Norske anførselstegn: **« »** (vinkeltegn)
- Alternative: **" "**

```
✅ «Prosjektet ditt 'Min bok' er lagret»
```

### Tall og formater

| Type         | Format     | Eksempel   |
| ------------ | ---------- | ---------- |
| Tusentall    | Mellomrom  | 1 000      |
| Desimaler    | Komma      | 3,14       |
| Valuta (NOK) | X kr       | 99 kr      |
| Valuta (€)   | X €        | 12,99 €    |
| Prosent      | X %        | 50 %       |
| Dato         | DD.MM.ÅÅÅÅ | 30.12.2025 |
| Tid          | HH:MM      | 14:30      |

### Spesielle bokstaver

Norsk har tre ekstra bokstaver:

- **Æ, Ø, Å** — plassert sist i alfabetet
