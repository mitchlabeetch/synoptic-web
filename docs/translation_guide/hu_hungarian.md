## 1. Hangnem és Regiszter

### A Synoptic Studio számára

- **Regiszter**: Professzionális, de barátságos
- **Hangnem**: Magabiztos, világos és segítőkész — soha nem lekezelő
- **Stílus**: Aktív előnyben; passzív csak a cselekvés hangsúlyozására

### Alapelvek

```
✅ «Mentsd el a projektedet» (közvetlen, aktív)
❌ «A projekted el lesz mentve» (passzív)

✅ «Készíts gyönyörű kétnyelvű könyveket»
```

---

## 2. Te vs. Ön

### Szabály a Synoptic számára

**Használd a «te» formát** — modern tech alkalmazások standardja.

| Kontextus              | Forma  | Példa                        |
| ---------------------- | ------ | ---------------------------- |
| Felhasználói felület   | **Te** | «Mentsd el a projektedet»    |
| Technikai dokumentáció | **Te** | «Exportálhatsz PDF-be»       |
| Hibaüzenetek           | **Te** | «Ellenőrizd a kapcsolatodat» |
| Jogi szövegek          | **Ön** | «A felhasználó elfogadja...» |

### ⚠️ Megjegyzés

- «Te» informális de barátságos — tökéletes tech alkalmazásokhoz
- «Ön» nagyon formális — csak jogi/hivatalos szövegekhez
- Ne keverd a formákat ugyanabban a dokumentumban

## 3. Nemtelen Nyelvhasználat

### Kihívás a magyarban

A magyar nyelvben nincs nyelvtani nem! Ez természetes előny.

| Névmás | Használat             |
| ------ | --------------------- |
| **Ő**  | Minden nemre ✅       |
| **Ők** | Többes szám, semleges |

### Legjobb gyakorlatok

```
✅ «Üdvözlünk!» (semleges)
✅ «A felhasználó mentheti a projektjét»

✅ «Szerzőknek és szerkesztőknek»
✅ «Alkotóknak»
```

## 5. Nyelvtan és Szintaxis

### Mondatszerkezet

A magyar **SOV** (Alany-Tárgy-Ige) vagy **szabad szórendű**:

```
Angol: I save the project (SVO)
Magyar: Én mentem a projektet (SVO — de változhat)
Magyar: A projektet mentem (számomra = hangsúly a projekten)
```

### Ragozás

A magyar agglutináló nyelv — ragok és képzők halmozódnak:

| Alapszó | Ragozott                            |
| ------- | ----------------------------------- |
| projekt | projekt**em** (az enyém)            |
| projekt | projekt**ed** (a tiéd)              |
| projekt | projekt**jét** (az övét, tárgyeset) |

### Magánhangzó-harmónia

A ragok illeszkednek a szótő hangrendjéhez:

- **Mély**: a, á, o, ó, u, ú → -ban, -ból
- **Magas**: e, é, i, í, ö, ő, ü, ű → -ben, -ből

## 6. Tipográfia

### Idézőjelek

- Magyar idézőjelek: **„ "** (lent-fent)

```
✅ „A projekted «Könyvem» mentve lett"
```

### Számok és formátumok

| Típus        | Formátum      | Példa         |
| ------------ | ------------- | ------------- |
| Ezres        | Szóköz        | 1 000         |
| Tizedes      | Vessző        | 3,14          |

| Pénznem (€)  | X €           | 12,99 €       |
| Százalék     | X%            | 50%           |
| Dátum        | ÉÉÉÉ. MM. DD. | 2025. 12. 30. |
| Idő          | HH:MM         | 14:30         |

### Magyar dátumformátum

A magyar dátum sorrend: **év. hónap. nap.**
