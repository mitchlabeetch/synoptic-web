## 1. Nada dan Register

### Untuk Synoptic Studio

- **Register**: Profesional tapi ramah
- **Nada**: Percaya diri, jelas, dan membantu — tidak pernah sombong
- **Suara**: Aktif lebih diutamakan; pasif hanya untuk menekankan tindakan

### Prinsip Utama

```
✅ «Simpan proyek Anda» (langsung, aktif)
❌ «Proyek Anda akan disimpan» (pasif)
```

## 2. Kata Ganti

### Aturan untuk Synoptic

**Gunakan «Anda»** — standar untuk aplikasi profesional.

| Konteks   | Kata Ganti    | Contoh                          |
| --------- | ------------- | ------------------------------- |
| UI umum   | **Anda**      | «Simpan proyek Anda»            |
| Error     | **Anda**      | «Silakan periksa koneksi Anda»  |
| Pemasaran | **Anda/kamu** | «Mulai perjalanan kreatif Anda» |

### Hierarki Kata Ganti

| Kata Ganti    | Tingkat       | Penggunaan di UI        |
| ------------- | ------------- | ----------------------- |
| **Saya**      | Netral        | ✅ Orang pertama        |
| **Anda**      | Sopan, formal | ✅ **Direkomendasikan** |
| **Kamu**      | Informal      | △ Pemasaran kasual      |
| **Bapak/Ibu** | Sangat formal | Dokumen resmi           |

## 3. Bahasa Netral Gender

### Keuntungan Bahasa Indonesia

Bahasa Indonesia **tidak memiliki gender gramatikal** — secara alami netral!

```
✅ «Pengguna dapat menyimpan proyek»
✅ «Penulis dan editor»
```

### Kata Ganti Orang Ketiga

- «Dia» — untuk semua gender ✅ **Netral**
- «Mereka» — jamak, netral

## 5. Tata Bahasa

### Struktur Kalimat

Bahasa Indonesia mengikuti **SVO** (Subjek-Verba-Objek):

```
Inggris: I save the project (SVO)
Indonesia: Saya menyimpan proyek (SVO) ✅ Sama!
```

### Afiks

Bahasa Indonesia menggunakan awalan dan akhiran:

- **me-** + simpan → menyimpan (to save)
- **di-** + simpan → disimpan (is saved)
- **pe-** + simpan → penyimpanan (saving/storage)
