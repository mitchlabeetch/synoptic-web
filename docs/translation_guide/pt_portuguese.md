## 1. Tom e Registro

### Para Synoptic Studio

- **Registro**: Profissional mas acessível
- **Tom**: Confiante, claro e prestativo — nunca condescendente
- **Voz**: Ativa preferida; passiva apenas para enfatizar a ação

### Princípios-chave

```
✅ «Guarde o seu projeto» / «Salve seu projeto»
❌ «O seu projeto será guardado»

✅ «Crie belos livros bilíngues»
❌ «Livros bilíngues podem ser criados por você»
```

## 2. Português Brasileiro vs. Europeu

### Principais diferenças

| Aspecto                    | Português Europeu (PT-PT) | Português Brasileiro (PT-BR) |
| -------------------------- | ------------------------- | ---------------------------- |
| Pronome                    | Tu                        | Você                         |
| Gerúndio                   | A guardar                 | Guardando                    |
| Artigo antes de possessivo | O meu projeto             | Meu projeto                  |
| Colocação pronominal       | Dá-me                     | Me dá                        |
| Vocabulário                | Telemóvel                 | Celular                      |
| Vocabulário                | Ecrã                      | Tela                         |
| Vocabulário                | Ficheiro                  | Arquivo                      |

### Recomendação para Synoptic

**Escolher uma variante** e manter consistência:

- **PT-BR** para mercado brasileiro (maior público)
- **PT-PT** para mercado europeu/africano

### Exemplos comparativos

| Contexto            | PT-BR               | PT-PT            |
| ------------------- | ------------------- | ---------------- |
| Botão Save          | Salvar              | Guardar          |
| Botão Delete        | Excluir             | Eliminar         |
| «You are logged in» | Você está conectado | Está autenticado |
| «Uploading...»      | Enviando...         | A enviar...      |
| «Settings»          | Configurações       | Definições       |

## 3. Tratamento: Tu vs. Você

### Português Brasileiro

- **Você** é o padrão universal
- **Tu** usado regionalmente (Sul do Brasil) mas conjugado como você

```
✅ PT-BR: « Você pode exportar seu projeto »
✅ PT-BR informal: « Você pode exportar o projeto » (sem possessivo)
```

### Português Europeu

- **Tu** para informal, **Você** para formal (mas cada vez mais informalmente)
- Terceira pessoa também usada formalmente

```
✅ PT-PT informal: « Podes exportar o teu projeto »
✅ PT-PT formal: « Pode exportar o seu projeto »
```

## 4. Linguagem Inclusiva

### Opções recomendadas

| Método                  | Exemplo                          | Uso                 |
| ----------------------- | -------------------------------- | ------------------- |
| **Reformulação neutra** | quem utiliza, pessoa utilizadora | ✅ Preferido        |
| **@ (arroba)**          | usuári@s                         | Evitar              |
| **x**                   | usuárixs                         | Evitar              |
| **e inclusivo**         | todes, amigues                   | Emergente, informal |

### Para Synoptic

```
✅ « Boas-vindas! » (neutro)
❌ « Bem-vindo/a »

✅ « Pessoas autoras e editoras »
✅ « Quem publica »
❌ « Os autores »
```

### Neutro sem marcação de gênero

Usar substantivos coletivos e reformulações:

```
✅ « A equipe editorial » (não « os editores »)
✅ « Profissionais de tradução » (não « tradutores »)
```

## 5. Falsos Amigos e Armadilhas

### Falsos amigos inglês-português

| Inglês        | ❌ Tradução errada | ✅ Tradução correta     |
| ------------- | ------------------ | ----------------------- |
| Actually      | Atualmente         | Na verdade, de fato     |
| Attend        | Atender            | Assistir, comparecer    |
| College       | Colégio            | Faculdade               |
| Comprehensive | Compreensivo       | Abrangente, completo    |
| Eventually    | Eventualmente      | Finalmente, por fim     |
| Exit          | Êxito              | Saída                   |
| Intend        | Entender           | Pretender, ter intenção |
| Library       | Livraria           | Biblioteca              |
| Push          | Puxar              | Empurrar                |
| Resume        | Resumir            | Retomar, continuar      |

### A armadilha «Ultimate»

Em inglês, «ultimate» significa **«o melhor, o definitivo»**.  
Em português, «último» significa **«o final»**, não necessariamente o melhor.

```
✅ « O estúdio bilíngue definitivo »
✅ « O melhor estúdio bilíngue »
❌ « O último estúdio bilíngue » (soa como "o final")
```

## 6. Gramática e Sintaxe

### Estrutura da frase

- **Sujeito-Verbo-Objeto (SVO)** é padrão
- Português permite **omitir o sujeito** quando claro

### Colocação pronominal

| PT-BR        | PT-PT        |
| ------------ | ------------ |
| Me envie     | Envie-me     |
| Te ajudo     | Ajudo-te     |
| Se encontram | Encontram-se |

### Gerúndio vs. Infinitivo

| PT-BR             | PT-PT             |
| ----------------- | ----------------- |
| Está salvando...  | Está a guardar... |
| Estou trabalhando | Estou a trabalhar |

## 7. Tipografia

### Aspas

- Aspas latinas preferidas: **« »**
- Alternativa válida: **" "** (aspas inglesas)

```
✅ « O seu projeto "Meu livro" foi guardado »
```

### Números e formatos

| Tipo        | Formato        | Exemplo        |
| ----------- | -------------- | -------------- |
| Milhares    | Ponto          | 1.000          |
| Decimais    | Vírgula        | 3,14           |
| Moeda (€)   | X,XX €         | 12,99 €        |
| Moeda (R$)  | R$ X,XX        | R$ 12,99       |
| Percentagem | X%             | 50%            |
| Data        | DD/MM/AAAA     | 30/12/2025     |
| Hora        | HH:MM ou HHhMM | 14:30 ou 14h30 |

## 8. Padrões de Interface

### Botões

- Usar **infinitivo** ou **imperativo**
- Máximo 1-3 palavras

| Ação     | PT-BR    | PT-PT           |
| -------- | -------- | --------------- |
| Save     | Salvar   | Guardar         |
| Export   | Exportar | Exportar        |
| Delete   | Excluir  | Eliminar        |
| Cancel   | Cancelar | Cancelar        |
| Create   | Criar    | Criar           |
| Edit     | Editar   | Editar          |
| Sign in  | Entrar   | Iniciar sessão  |
| Sign out | Sair     | Terminar sessão |

### Mensagens de erro

```
✅ PT-BR: « Conexão perdida. Verifique sua internet e tente novamente. »
✅ PT-PT: « Ligação perdida. Verifique a sua ligação à Internet e tente novamente. »
❌ « Erro de rede »
```

## 10. Acordo Ortográfico

O Acordo Ortográfico de 1990 unificou algumas grafias:

- **Sem hífen** em prefixos comuns: autoestima, antirracismo
- **Eliminação de consoantes mudas** em PT-PT: ação (não acção), diretor (não director)

Ambas variantes devem seguir o Acordo Ortográfico atualizado.
