## 1. Tón a Registr

### Pro Synoptic Studio

- **Registr**: Profesionální, ale přístupný
- **Tón**: Sebevědomý, jasný a nápomocný — nikdy povýšený
- **Hlas**: Aktivní preferován; pasivní pouze pro zdůraznění akce

### Klíčové principy

```
✅ «Ulož svůj projekt» (přímý, aktivní)
❌ «Tvůj projekt bude uložen» (pasivní)

✅ «Vytvoř krásné dvojjazyčné knihy»
```

---

## 2. Ty vs. Vy

### Pravidlo pro Synoptic

**Používej «ty»** — standard pro moderní technologické aplikace.

| Kontext               | Forma  | Příklad                    |
| --------------------- | ------ | -------------------------- |
| Uživatelské rozhraní  | **Ty** | «Ulož svůj projekt»        |
| Technická dokumentace | **Ty** | «Můžeš exportovat do PDF»  |
| Chybové zprávy        | **Ty** | «Zkontroluj své připojení» |
| Právní texty          | **Vy** | «Uživatel souhlasí...»     |

### ⚠️ Poznámka

- «Ty» je neformální, ale přátelské — perfektní pro tech
- «Vy» je velmi formální — pouze pro právní/oficiální texty

---

## 3. Genderově neutrální jazyk

### Výzva v češtině

Čeština má gramatický rod (mužský, ženský, střední). Slovesa a přídavná jména se shodují.

### Doporučené strategie

| Metoda                        | Příklad            | Použití          |
| ----------------------------- | ------------------ | ---------------- |
| **Neosobní konstrukce**       | «Lze exportovat»   | ✅ Preferováno   |
| **Množné číslo**              | «Uživatelé mohou»  | ✅ Neutrální     |
| **Neutrální podstatná jména** | «Osoba»            | Pro právní texty |
| **Dvojí tvary**               | «Autoři a autorky» | Inkluzivní       |

### Pro Synoptic

```
✅ «Vítej!» (neutrální kontextem)
✅ «Projekt byl uložen» (pasivní — neutrální)
❌ «Vítej, milý uživateli»

✅ «Pro tvůrce a editory»
```

## 4. Překladové pasti

### Falešní přátelé

| Angličtina  | ❌ Špatný překlad | ✅ Správný překlad |
| ----------- | ----------------- | ------------------ |
| Actual      | Aktuální          | Skutečný           |
| Eventually  | Eventuálně        | Nakonec            |
| Sympathetic | Sympatický        | Soucitný           |
| Concrete    | Konkrétní         | Beton (materiál)   |

### Past «Ultimate»

V angličtině «ultimate» často znamená **«nejlepší, ultimátní»**.

```
✅ «Ultimátní dvojjazyčné studio»
✅ «Nejlepší dvojjazyčné studio»
❌ «Poslední dvojjazyčné studio» (naznačuje konec)
```

## 5. Gramatika a Syntax

### Struktura věty

Čeština má relativně **volný slovosled** díky pádům:

```
Angličtina: I save the project (SVO)
Čeština: Já ukládám projekt (SVO)
Čeština: Projekt ukládám (OVS — důraz na projekt)
```

### Pádový systém

Čeština má 7 pádů:

| Pád          | Otázka        | Příklad      |
| ------------ | ------------- | ------------ |
| Nominativ    | Kdo? Co?      | projekt      |
| Genitiv      | Koho? Čeho?   | projektu     |
| Dativ        | Komu? Čemu?   | projektu     |
| Akuzativ     | Koho? Co?     | projekt      |
| Vokativ      | —             | projektu!    |
| Lokál        | O kom? O čem? | (o) projektu |
| Instrumentál | Kým? Čím?     | projektem    |

## 6. Typografie

### Uvozovky

- České uvozovky: **„ "** (dole-nahoře)

```
✅ „Tvůj projekt ‚Moje kniha' byl uložen"
```

### Čísla a formáty

| Typ       | Formát       | Příklad      |
| --------- | ------------ | ------------ |
| Tisíce    | Mezera       | 1 000        |
| Desetinná | Čárka        | 3,14         |
| Měna (Kč) | X Kč         | 299 Kč       |
| Měna (€)  | X €          | 12,99 €      |
| Procenta  | X %          | 50 %         |
| Datum     | DD. MM. RRRR | 30. 12. 2025 |
| Čas       | HH:MM        | 14:30        |

### Diakritika

Vždy používej diakritiku:

- **á, č, ď, é, ě, í, ň, ó, ř, š, ť, ú, ů, ý, ž**

```
✅ «Uložit»
❌ «Ulozit» (bez háčků a čárek)
```
