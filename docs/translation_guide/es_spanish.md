## 1. Tono y Registro

### Para Synoptic Studio

- **Registro**: Profesional pero accesible
- **Tono**: Seguro, claro y servicial — nunca condescendiente
- **Voz**: Activa preferida; pasiva solo para enfatizar la acción

### Principios clave

```
✅ «Guarda tu proyecto» (directo, activo)
❌ «Tu proyecto será guardado» (pasivo)

✅ «Crea hermosos libros bilingües»
❌ «Los libros bilingües pueden ser creados por ti»
```

## 2. Tuteo vs. Usted

### Regla para Synoptic

**Utilice el tuteo (tú)** para la interfaz — es el estándar moderno para aplicaciones tecnológicas en la mayoría de mercados.

| Contexto              | España       | Latinoamérica | Recomendación     |
| --------------------- | ------------ | ------------- | ----------------- |
| UI informal           | **Tú**       | **Tú**        | ✅ Usar tú        |
| Documentación técnica | **Tú/Usted** | **Usted**     | Tú preferido      |
| Legal/Términos        | **Usted**    | **Usted**     | Usted obligatorio |
| Marketing             | **Tú**       | **Tú**        | Tú para cercanía  |

### ⚠️ Voseo (Argentina, Uruguay, partes de Centroamérica)

El voseo es **opcional** — usar solo si se localiza específicamente para esos mercados.

```
Tuteo estándar: « Guarda tu proyecto »
Voseo: « Guardá tu proyecto » (Argentina)
```

### Usted para contextos formales

```
✅ « Al aceptar, usted confirma haber leído los términos »
```

### Opciones recomendadas

| Método                 | Ejemplo                         | Uso                     |
| ---------------------- | ------------------------------- | ----------------------- |

| **Arroba/x**           | usuari@s, usuarixs              | Evitar (no legible)     |

```

✅ « Personas autoras y editoras »
✅ « Quienes publican »
❌ « Los autores »

✅ « Tu equipo de trabajo »
❌ « Tus compañeros »
```

### Masculino genérico con matiz

Cuando el contexto lo requiere, se puede usar el masculino genérico con una nota:

```
« A lo largo de este documento, el término "usuario" incluye a todas las personas sin distinción de género. »
```

## 4. Falsos Amigos y Trampas

### Falsos amigos inglés-español

| Inglés        | ❌ Mala traducción       | ✅ Buena traducción             |
| ------------- | ------------------------ | ------------------------------- |
| Actually      | Actualmente              | En realidad, de hecho           |
| Argument      | Argumento (pelea)        | Argumento (razón) ✓ / Discusión |
| Assist        | Asistir (estar presente) | Ayudar                          |
| Casual        | Casual (por casualidad)  | Informal                        |
| Comprehensive | Comprensivo              | Exhaustivo, completo            |
| Eventually    | Eventualmente            | Finalmente, a la larga          |
| Library       | Librería                 | Biblioteca                      |
| Realize       | Realizar                 | Darse cuenta                    |
| Resume        | Resumir                  | Reanudar, continuar             |
| Sensible      | Sensible                 | Sensato, razonable              |
| Success       | Suceso                   | Éxito                           |

### La trampa «Ultimate»

En inglés, «ultimate» significa **«el mejor, el definitivo»**.  
En español, «último» significa **«el final»**, no necesariamente el mejor.

```
✅ « El estudio bilingüe definitivo »
✅ « El mejor estudio bilingüe »
❌ « El último estudio bilingüe » (suena a "el final")
```

## 5. Gramática y Sintaxis

### Estructura de oración

- **Sujeto-Verbo-Objeto (SVO)** es estándar pero flexible
- El español permite **omitir el sujeto** cuando es claro:
  - ✅ « Guarda tu proyecto » (sujeto implícito: tú)

### Adjetivos

- Generalmente **después del sustantivo**: « un libro bilingüe »
- Algunos antes con significado diferente:
  - « un gran proyecto » (excelente) vs. « un proyecto grande » (tamaño)

### Acentos obligatorios

Los acentos cambian el significado — nunca omitirlos:

| Con acento      | Sin acento                            |
| --------------- | ------------------------------------- |
| Él (pronombre)  | El (artículo)                         |
| Sí (afirmación) | Si (condicional)                      |
| Más (cantidad)  | Mas (pero)                            |
| Sólo/Solo       | Solo (adjetivo/adverbio, nueva norma) |

## 6. Tipografía Española

### Signos de puntuación invertidos

El español usa signos de apertura **¿** y **¡**:

```
✅ « ¿Guardar cambios? »
❌ « Guardar cambios? »

✅ « ¡Proyecto guardado! »
❌ « Proyecto guardado! »
```

### Comillas

- Comillas españolas preferidas: **« »** (angulares)
- Alternativa válida: **" "** (inglesas)
- Comillas simples para citas dentro de citas: **' '**

```
✅ « Tu proyecto "Mi libro" se ha guardado »
```

### Números y formatos

| Tipo       | Formato         | Ejemplo          |
| ---------- | --------------- | ---------------- |
| Miles      | Punto o espacio | 1.000 o 1 000    |
| Decimales  | Coma            | 3,14             |
| Moneda (€) | X,XX €          | 12,99 €          |
| Moneda ($) | $X.XX o X,XX $  | $12.99 o 12,99 $ |
| Porcentaje | X %             | 50 %             |
| Fecha      | DD/MM/AAAA      | 30/12/2025       |
| Hora       | HH:MM           | 14:30            |

## 7. Diferencias Regionales

### España vs. Latinoamérica

| Término            | España   | Latinoamérica              |
| ------------------ | -------- | -------------------------- |
| Ordenador          | ✅       | Computadora                |
| Móvil              | ✅       | Celular                    |
| Correo electrónico | ✅       | E-mail más común           |
| Vale               | ✅       | OK, de acuerdo             |
| Coger              | ✅ Tomar | ⚠️ Vulgar en muchos países |
| Conducir           | ✅       | Manejar                    |

### Palabras a evitar

**«Coger»** tiene connotación vulgar en gran parte de Latinoamérica. Usar **«tomar»** o **«agarrar»**.

```
✅ « Toma este libro » (universal)
❌ « Coge este libro » (problemático en LATAM)
```

### Para Synoptic

- Usar **español neutro/internacional** cuando sea posible
- Evitar regionalismos fuertes
- Si hay duda, preferir términos latinoamericanos (mercado más grande)

## 8. Patrones de Interfaz

### Botones

- Usar **infinitivo** o **imperativo (tú)**
- Máximo 1-3 palabras

| Acción   | Español        |
| -------- | -------------- |
| Save     | Guardar        |
| Export   | Exportar       |
| Delete   | Eliminar       |
| Cancel   | Cancelar       |
| Create   | Crear          |
| Edit     | Editar         |
| Sign in  | Iniciar sesión |
| Sign out | Cerrar sesión  |

### Mensajes de error

- Sea **específico** sobre el problema
- Ofrezca una **solución**
- No culpe al usuario

```
✅ « Conexión perdida. Comprueba tu conexión a internet e inténtalo de nuevo. »
❌ « Error de red »
❌ « Has perdido la conexión »
```

## 10. Consideraciones Culturales

### Formalidad

- El español tech moderno prefiere el tuteo
- Los términos legales requieren usted
- Evitar excesiva formalidad («estimado usuario»)

### Expresiones a adaptar

No traducir literalmente expresiones idiomáticas:

| Inglés                     | ❌ Literal           | ✅ Adaptado              |
| -------------------------- | -------------------- | ------------------------ |
| « Piece of cake »          | « Pedazo de pastel » | « Pan comido »           |
| « Hit the ground running » | (sin sentido)        | « Empezar con buen pie » |
