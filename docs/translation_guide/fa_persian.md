# فارسی (fa) — راهنمای ترجمه

## نمای کلی

| ویژگی             | مقدار                             |
| ----------------- | --------------------------------- |
| **کد ISO**        | `fa`                              |
| **نام بومی**      | فارسی / پارسی                     |
| **سیستم نوشتاری** | الفبای فارسی (مبتنی بر عربی)      |
| **جهت**           | راست به چپ (RTL)                  |
| **مناطق اصلی**    | ایران، افغانستان (دری)، تاجیکستان |

---

## 1. جهت متن (RTL)

### قوانین اساسی

- متن فارسی از **راست به چپ** نوشته می‌شود
- اعداد از چپ به راست خوانده می‌شوند
- رابط کاربری باید **آینه‌ای** (mirrored) باشد

### تأثیر بر طراحی

```
✅ دکمه‌ها و منوها در سمت راست
✅ نوار پیمایش در سمت چپ
✅ فلش‌ها معکوس (→ تبدیل به ←)
```

---

## 2. سطح خطاب

### برای Synoptic

**از «شما» استفاده کنید** — استاندارد برای برنامه‌های حرفه‌ای.

| زمینه        | شکل        | مثال                            |
| ------------ | ---------- | ------------------------------- |
| رابط کاربری  | **شما**    | «پروژه خود را ذخیره کنید»       |
| پیام‌های خطا | **شما**    | «لطفاً اتصال خود را بررسی کنید» |
| بازاریابی    | **شما/تو** | «سفر خلاقیت خود را آغاز کنید»   |

### ⚠️ توجه

- «شما» رسمی و محترمانه است
- «تو» غیررسمی است — فقط برای بازاریابی خودمانی

---

## 3. زبان بی‌طرف جنسیتی

### ویژگی فارسی

فارسی **جنس دستوری ندارد** — به طور طبیعی بی‌طرف است!

| ضمیر     | کاربرد               |
| -------- | -------------------- |
| **او**   | برای همه جنسیت‌ها ✅ |
| **آنها** | جمع، بی‌طرف          |

```
✅ «کاربر می‌تواند پروژه خود را ذخیره کند»
✅ «نویسندگان و ویراستاران»
```

---

## 4. دام‌های ترجمه

### ترجمه «Ultimate»

در انگلیسی، «ultimate» به معنای **«بهترین، نهایی»** است.

```
✅ «استودیوی دوزبانه نهایی»
✅ «بهترین استودیوی دوزبانه»
❌ «آخرین استودیوی دوزبانه» (پایان را تداعی می‌کند)
```

### اصطلاحات فنی

| File      | فایل / پرونده |

## 5. اعداد فارسی

### انتخاب سیستم اعداد

| نوع         | شکل        |
| ----------- | ---------- |
| اعداد فارسی | ۰۱۲۳۴۵۶۷۸۹ |
| اعداد غربی  | 0123456789 |

**توصیه:** برای برنامه‌های فنی از اعداد غربی (0-9) استفاده کنید.

### قالب‌ها

| نوع              | قالب       | مثال         |
| ---------------- | ---------- | ------------ |
| هزارگان          | ویرگول     | 1,000        |
| اعشار            | نقطه       | 3.14         |
| واحد پول (ریال)  | X ریال     | 499,000 ریال |
| واحد پول (تومان) | X تومان    | 49,900 تومان |
| درصد             | X%         | 50%          |
| تاریخ شمسی       | YYYY/MM/DD | 1404/10/10   |
| تاریخ میلادی     | DD/MM/YYYY | 30/12/2025   |

## 6. تایپوگرافی

### گیومه‌ها

- گیومه‌های فارسی: **« »**

```
✅ «پروژه شما «کتاب من» ذخیره شد»
```

### نویسه‌های خاص

فارسی دارای حروف اضافی نسبت به عربی است:

- **پ، چ، ژ، گ**
