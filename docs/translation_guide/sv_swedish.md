## 1. Ton och Register

### För Synoptic Studio

- **Register**: Professionellt men tillgängligt
- **Ton**: Självförtroende, tydlig och hjälpsam — aldrig nedlåtande
- **Röst**: Aktiv föredras; passiv endast för att betona handlingen

### Grundprinciper

```
✅ «Spara ditt projekt» (direkt, aktivt)
❌ «Ditt projekt kommer att sparas» (passivt)

✅ «Skapa vackra tvåspråkiga böcker»
❌ «Tvåspråkiga böcker kan skapas av dig»
```

## 2. Du vs. Ni

### Regel för Synoptic

**Använd «du»** — det är standard i modern svensk kommunikation.

| Kontext               | Form       | Exempel                      |
| --------------------- | ---------- | ---------------------------- |
| Användargränssnitt    | **Du**     | «Spara ditt projekt»         |
| Teknisk dokumentation | **Du**     | «Du kan exportera till PDF»  |
| Felmeddelanden        | **Du**     | «Kontrollera din anslutning» |
| Juridiska texter      | **Ni/Man** | «Användaren godkänner...»    |

### ⚠️ Historisk kontext: Du-reformen

- På 1960-talet genomgick Sverige «du-reformen»
- Före reformen var «ni» formellt, men det uppfattades ofta som nedlåtande
- Idag är «du» standardformen i alla sammanhang
- «Ni» används sällan och kan uppfattas som konstigt eller ålderdomligt

## 3. Könsneutralt Språk

### Rekommenderade strategier

| Metod             | Exempel                            | Användning       |
| ----------------- | ---------------------------------- | ---------------- |
| **Hen**           | «Användaren och hens projekt»      | ✅ Rekommenderat |
| **De/Dem**        | «Användare och deras projekt»      | ✅ Naturligt     |
| **Omformulering** | «Den som använder»                 | ✅ Elegant       |
| **Plural**        | «Användare kan spara sina projekt» | ✅ Praktiskt     |

### Pronomen «hen»

Sverige introducerade officiellt «hen» som könsneutralt pronomen:

- **Han** = he
- **Hon** = she
- **Hen** = de/dem (singular, könsneutralt) ✅

### Exempel

```
✅ «Välkommen!» (neutralt)
✅ «Användaren kan spara sitt projekt» (possessivt «sitt» är neutralt)
✅ «Hen kan exportera sina filer»

✅ «För författare och redaktörer»
❌ «Välkommen, kära användare» (onödigt formellt)
```

### «Ultimate»-fällan

På engelska betyder «ultimate» ofta **«bästa, ultimata»**.
På svenska fungerar «ultimat» bra i samma betydelse.

```
✅ «Den ultimata tvåspråkiga studion» — korrekt betydelse
✅ «Den bästa tvåspråkiga studion»
❌ «Den sista tvåspråkiga studion» (antyder slutet)
```

## 5. Grammatik och Syntax

### Meningsstruktur

Svenska följer generellt **SVO** (Subjekt-Verb-Objekt):

```
Engelska: I save the project (SVO)
Svenska: Jag sparar projektet (SVO)
```

### V2-regeln

I huvudsatser står verbet alltid på andra plats:

```
✅ «Nu kan du spara projektet» (Adverb-Verb-Subjekt-Objekt)
✅ «Projektet sparas automatiskt» (Objekt-Verb-Adverb)
```

### Bestämd och obestämd form

Svenska har suffix för bestämd form:

| Obestämd    | Bestämd       | Betydelse               |
| ----------- | ------------- | ----------------------- |
| en fil      | fil**en**     | a file / the file       |
| ett projekt | projekt**et** | a project / the project |
| filer       | filer**na**   | files / the files       |

## 6. Typografi

### Citattecken

- Svenska citattecken: **" "** eller **» «**
- Enkla: **' '**

```
✅ "Ditt projekt 'Min bok' har sparats"
```

### Siffror och format

| Typ          | Format               | Exempel    |
| ------------ | -------------------- | ---------- |
| Tusentals    | Mellanslag           | 1 000      |
| Decimaler    | Komma                | 3,14       |
| Valuta (SEK) | X kr                 | 129 kr     |
| Valuta (€)   | X €                  | 12,99 €    |
| Procent      | X % (med mellanslag) | 50 %       |
| Datum        | ÅÅÅÅ-MM-DD           | 2025-12-30 |
| Tid          | HH:MM                | 14:30      |

### Särskilda bokstäver

Svenska har tre extra bokstäver:

- **Å, Ä, Ö** — placeras sist i alfabetet

## 9. Kulturella överväganden

### Lagom och tydlighet

- Svenska användare föredrar koncis, tydlig kommunikation
- Undvik överdriven formalitet — det känns konstlat
- «Lagom» (inte för mycket, inte för lite) är en svensk kulturell princip

### Jämlikhet

- Sverige är ett jämlikt samhälle — undvik hierarkiskt språk
- «Du» till alla, oavsett position
- Könsneutralt språk förväntas
