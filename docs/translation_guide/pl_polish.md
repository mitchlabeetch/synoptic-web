## 1. Ton i Rejestr

### Dla Synoptic Studio

- **Rejestr**: Profesjonalny, ale przystępny
- **Ton**: Pewny siebie, jasny i pomocny — nigdy protekcjonalny
- **Głos**: Strona czynna preferowana; bierna tylko dla podkreślenia działania

### Kluczowe zasady

```
✅ «Zapisz swój projekt» (bezpośredni, aktywny)
❌ «Twój projekt zostanie zapisany» (bierny)

✅ «Twórz piękne książki dwujęzyczne»
❌ «Książki dwujęzyczne mogą być tworzone przez Ciebie»
```

## 2. Ty vs. Pan/Pani

### Zasada dla Synoptic

**Używaj formy „ty"** — to standard dla nowoczesnych aplikacji technologicznych.

| Kontekst                | Forma        | Przykład                               |
| ----------------------- | ------------ | -------------------------------------- |
| Interfejs użytkownika   | **Ty**       | «Zapisz swój projekt»                  |
| Dokumentacja techniczna | **Ty**       | «Możesz wyeksportować do PDF»          |
| Komunikaty o błędach    | **Ty**       | «Sprawdź swoje połączenie»             |
| Teksty prawne           | **Pan/Pani** | «Użytkownik akceptuje...» / bezosobowo |

### Pisownia zaimków

- **Skrócone formy** (ty, twój, ciebie) — małą literą w większości kontekstów
- **Wielka litera** (Ty, Twój) — w korespondencji bezpośredniej jako wyraz szacunku (opcjonalnie w UI)

```
✅ « Zapisz swój projekt » (standardowo)
✅ « Zapisz Swój projekt » (bardziej formalnie, ale rzadko w UI)
```

## 3. Język Inkluzywny

### Wyzwania w polskim

Polski jest językiem silnie fleksyjnym z wyraźnym podziałem na rodzaje (męski, żeński, nijaki). Inkluzywność jest trudniejsza niż w angielskim.

### Zalecane strategie

| Metoda                                       | Przykład                            | Zastosowanie        |
| -------------------------------------------- | ----------------------------------- | ------------------- |
| **Formy bezosobowe**                         | «Można eksportować»                 | ✅ Preferowane      |
| **Formy rodzaju męskoosobowego z adnotacją** | «Użytkownicy (w tym użytkowniczki)» | Formalne            |
| **Podwójne formy**                           | «Autorzy i autorki»                 | Gdy miejsce pozwala |
| **Neutralne rzeczowniki**                    | «Osoby piszące»                     | ✅ Eleganckie       |

### Dla Synoptic

```
✅ « Witaj! » (neutralne)
❌ « Witaj, drogi użytkowniku »

✅ « Dla osób tworzących i publikujących »
❌ « Dla autorów »

✅ « Twój zespół »
❌ « Twoi współpracownicy »
```

### Formy zwrotne jako rozwiązanie

```
✅ « Projekt został zapisany » (bezosobowe)
✅ « Możesz tworzyć książki » (2. osoba, neutralne płciowo)
```

## 4. Fałszywi Przyjaciele i Pułapki

### Fałszywi przyjaciele angielsko-polscy

| Angielski    | ❌ Błędne tłumaczenie | ✅ Prawidłowe tłumaczenie       |
| ------------ | --------------------- | ------------------------------- |
| Actual       | Aktualny              | Faktyczny, rzeczywisty          |
| Eventually   | Ewentualnie           | Ostatecznie, w końcu            |
| Fabric       | Fabryka               | Tkanina, materiał               |
| Lunatic      | Lunatyk               | Szaleniec                       |
| Sympathetic  | Sympatyczny           | Współczujący, wyrozumiały       |
| Preservative | Prezerwatywa          | Konserwant                      |
| Concrete     | Konkretny             | Beton / Konkretny (obie formy!) |

### Pułapka «Ultimate»

Po angielsku «ultimate» często oznacza **«najlepszy, ostateczny»**.  
Po polsku «ultimatywny» ma bardziej formalny i ostrzegawczy wydźwięk (ultimatum).

```
✅ « Najlepsze studio dwujęzyczne »
✅ « Kompletne studio dwujęzyczne »
❌ « Ultimatywne studio dwujęzyczne » (brzmi ostrzegawczo)
```

## 5. Gramatyka i Składnia

### Struktura zdania

- **Podmiot-Orzeczenie-Dopełnienie (POD)** jest standardem, ale szyk jest elastyczny
- Końcówki fleksyjne pozwalają na swobodniejszy szyk wyrazów

### Deklinacja

Polski ma 7 przypadków — ważne dla poprawności:

| Przypadek   | Pytanie        | Przykład      |
| ----------- | -------------- | ------------- |
| Mianownik   | Kto? Co?       | projekt       |
| Dopełniacz  | Kogo? Czego?   | projektu      |
| Celownik    | Komu? Czemu?   | projektowi    |
| Biernik     | Kogo? Co?      | projekt       |
| Narzędnik   | Kim? Czym?     | projektem     |
| Miejscownik | O kim? O czym? | (o) projekcie |
| Wołacz      | —              | projekcie!    |

### Przykłady

```
✅ « Zapisz swój projekt » (biernik)
✅ « Ustawienia projektu » (dopełniacz)
✅ « Pracuj nad projektem » (narzędnik)
```

## 6. Typografia

### Cudzysłowy

- Polskie cudzysłowy: **„ "** (podwójne polskie)
- Alternatywa: **» «** (ostrokątne)

```
✅ „Twój projekt ‚Moja książka' został zapisany"
```

### Liczby i formaty

| Typ               | Format     | Przykład   |
| ----------------- | ---------- | ---------- |
| Tysiące           | Spacja     | 1 000      |
| Ułamki dziesiętne | Przecinek  | 3,14       |
| Waluta (€)        | X,XX €     | 12,99 €    |
| Waluta (zł)       | X,XX zł    | 12,99 zł   |
| Procent           | X%         | 50%        |
| Data              | DD.MM.RRRR | 30.12.2025 |
| Godzina           | HH:MM      | 14:30      |

### Polskie znaki

Pamiętaj o wszystkich polskich znakach diakrytycznych:
**ą, ć, ę, ł, ń, ó, ś, ź, ż**

```
✅ « Książka została zapisana »
❌ « Ksiazka zostala zapisana » (brak polskich znaków)
```

## 7. Wzorce Interfejsu

### Przyciski

- Użyj **trybu rozkazującego** (2. osoba l.poj.)
- Maksymalnie 1-3 słowa

| Akcja    | Polski      |
| -------- | ----------- |
| Save     | Zapisz      |
| Export   | Eksportuj   |
| Delete   | Usuń        |
| Cancel   | Anuluj      |
| Create   | Utwórz      |
| Edit     | Edytuj      |
| Sign in  | Zaloguj się |
| Sign out | Wyloguj się |

### Komunikaty o błędach

- Bądź **konkretny** co do problemu
- Zaproponuj **rozwiązanie**
- Nie obwiniaj użytkownika

```
✅ « Utracono połączenie. Sprawdź połączenie internetowe i spróbuj ponownie. »
❌ « Błąd sieci »
❌ « Straciłeś połączenie »
```

## 9. Uwagi Kulturowe

### Bezpośredniość

Polscy użytkownicy cenią:

- Jasny, bezpośredni język
- Unikanie zbyt kwiecistego stylu
- Poprawność językową

### Anglicyzmy

Polski przyjmuje anglicyzmy, ale często istnieją rodzime odpowiedniki:

- ✅ « Plik » zamiast « file »
- ✅ « Pobierz » zamiast « download »
- ⚠️ « Upload » → « Prześlij » lub « Wyślij »
