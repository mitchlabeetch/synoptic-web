# 中文繁體 (zh-TW) — 翻譯指南

## 概覽

| 屬性         | 值                  |
| ------------ | ------------------- |
| **ISO 代碼** | `zh-TW`             |
| **本地名稱** | 繁體中文 / 正體中文 |
| **文字系統** | 繁體漢字            |
| **書寫方向** | 從左到右 (LTR)      |
| **主要地區** | 台灣、香港、澳門    |

---

## 1. 語氣與語域

### Synoptic Studio 介面

- **語域**：專業但親切
- **語氣**：自信、清晰、有幫助——從不居高臨下
- **語態**：主動語態為主；被動語態僅用於強調動作

### 關鍵原則

```
✅ 「儲存您的專案」（直接、主動）
❌ 「您的專案將被儲存」（被動）

✅ 「建立精美的雙語圖書」
❌ 「雙語圖書可以由您建立」
```

---

## 2. 稱謂：你 vs. 您

### Synoptic 規則

**使用「您」** — 這是專業應用程式的標準。

| 情境       | 形式      | 範例                   |
| ---------- | --------- | ---------------------- |
| 使用者介面 | **您**    | 「儲存您的專案」       |
| 錯誤訊息   | **您**    | 「請檢查您的網路連線」 |
| 行銷       | **你/您** | 「開啟你的創作之旅」   |
| 法律文件   | **您**    | 「使用者同意...」      |

### 注意事項

- 「您」比「你」更正式、更尊重
- 在整個應用程式中保持一致
- 香港用戶也期望使用「您」

## 4. 翻譯陷阱

### 「Ultimate」的翻譯

在英文中，「ultimate」通常表示**「最好的，終極的」**。

```
✅ 「終極雙語工作室」
✅ 「頂級雙語創作平台」
❌ 「最後的雙語工作室」（暗示結束）
```

### 常見誤譯

| 英文      | ❌ 錯誤翻譯 | ✅ 正確翻譯    |
| --------- | ----------- | -------------- |
| Ultimate  | 最後的      | 終極的         |
| Save      | 拯救        | 儲存           |
| Character | 角色        | 字元（軟體）   |
| Library   | 圖書館      | 函式庫（軟體） |

### 技術術語

| 英文      | 繁體中文 |

## 5. 標點符號

使用全形中文標點：

| 符號   | 繁體中文 | 英文 |
| ------ | -------- | ---- |
| 句號   | 。       | .    |
| 逗號   | ，       | ,    |
| 問號   | ？       | ?    |
| 驚嘆號 | ！       | !    |
| 引號   | 「」     | ""   |
| 雙引號 | 『』     | ''   |
| 書名號 | 《》     | —    |
| 冒號   | ：       | :    |

## 9. 地區差異

### 台灣 vs. 香港

| 概念     | 台灣 | 香港 |
| -------- | ---- | ---- |
| Software | 軟體 | 軟件 |
| Hardware | 硬體 | 硬件 |
| Video    | 影片 | 視頻 |
| Network  | 網路 | 網絡 |

### Synoptic 建議

- 為台灣 (zh-TW) 和香港 (zh-HK) 分別考慮術語差異
- 使用台灣繁體作為預設

---

## 10. 文化注意事項

### 中英文混排

在中文和英文/數字之間添加空格：

```
✅ 「匯出為 PDF 格式」
✅ 「支援 30 多種語言」
❌ 「匯出為PDF格式」
```

### 本地化最佳實踐

- 注意顏色象徵意義
- 日期格式使用台灣標準
